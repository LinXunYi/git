## 1. 升降压拓扑概览

### 1.1 为什么需要升降压拓扑？

在实际系统中，输入电压范围可能跨越输出电压值。例如：

- 锂电池 3.0V~4.2V 供电，需要 3.3V 输出（VIN 可能 > VOUT 也可能 < VOUT）
- 12V 汽车系统（VIN 范围 6V~18V），需要 5V 和 12V 多路输出
- 太阳能板供电（VIN 随光照变化），需要稳定输出

Buck 只能在 VIN > VOUT 时工作，Boost 只能在 VIN < VOUT 时工作。**升降压拓扑可以处理 VIN 与 VOUT 的任何大小关系**。

### 1.2 常见升降压拓扑分类

| 拓扑 | 输出极性 | 开关数 | 电感数 | 特点 |
|------|---------|--------|--------|------|
| **反相 Buck-Boost** | 反相（负） | 2 | 1 | 最经典，结构简单 |
| **四开关 Buck-Boost** | 非反相 | 4 | 1 | 主流方案，TI/MPS 主推 |
| **SEPIC** | 非反相 | 2 | 2 | 输入电流连续，电容隔直 |
| **Ćuk** | 反相 | 2 | 2 | 输入输出电流均连续 |
| **Zeta** | 非反相 | 2 | 2 | 较少使用，SEPIC 变种 |

## 2. 经典反相 Buck-Boost 电感设计

### 2.1 拓扑结构

```
VIN ---+---[SW]---+---[L]---+
       |          |          |
      [Cin]      GND        [D]---+--- VOUT(-)
       |                      [C]   [RL]
       |                       |     |
GND ---+-----------------------+-----+

占空比：D = |VOUT| / (VIN + |VOUT|)
```

> 输出电压极性为负（相对于 VIN 的地）。

### 2.2 电感位置与电流

**电感连接在开关节点和输出之间**，这是 Buck-Boost 拓扑的独特之处：

- **开关导通时**：VIN 加在电感上，电流从 VIN 经 SW 流过 L 到地 → 储能
- **开关关断时**：电感通过二极管 D 向输出电容和负载释放能量

**电感直流电流**：

```
IL(dc) = IIN = IOUT × |VOUT| / (VIN × η)   （从输入侧看）
       = IOUT / (1 - D)                      （从电感侧看）

注意：这里的 IOUT 是流过负载的绝对电流值
```

**关键点**：反相 Buck-Boost 的电感直流电流 = IOUT / (1-D)

### 2.3 占空比公式

```
D = |VOUT| / (VIN + |VOUT|)
```

- VIN = VOUT 时：D = 0.5
- VIN < VOUT 时：D > 0.5（类似 Boost）
- VIN > VOUT 时：D < 0.5（类似 Buck）

### 2.4 电感值计算公式

在导通阶段，VIN 加在电感上：

```
VIN = L × ΔIL / ton → L = VIN × D / (ΔIL × fSW)
```

代入 D 表达式：

```
        VIN × |VOUT|
  L = ─────────────────────────────
        ΔIL × fSW × (VIN + |VOUT|)
```

用纹波系数 r = ΔIL / IL(dc) 表示：

```
        VIN × |VOUT|
  L = ──────────────────────────────────
        r × IL(dc) × fSW × (VIN + |VOUT|)
```

其中 IL(dc) = IOUT / (1-D) = IOUT × (VIN + |VOUT|) / VIN

代入化简：

```
        VIN²
  L = ──────────────────────────────────
        r × IOUT × fSW × (VIN + |VOUT|)
```

> ⚠️ **注意公式区别**：
> - Buck: L = VOUT×(VIN-VOUT) / (r×IOUT×fSW×VIN)
> - Boost: L = VIN×(VOUT-VIN) / (r×IIN×fSW×VOUT)
> - 反相 Buck-Boost: L = VIN² / (r×IOUT×fSW×(VIN+|VOUT|))

### 2.5 峰值电流

```
IPEAK = IL(dc) × (1 + r/2)
      = [IOUT/(1-D)] × (1 + r/2)
      = IOUT × (VIN+|VOUT|)/VIN × (1 + r/2)
```

```
Isat ≥ IPEAK × 1.2
Irms ≥ IL(dc) = IOUT / (1-D)
```

### 2.6 最大电感值条件

对 L 关于 VIN 求极值：

```
dL/dVIN = 0 → 需要具体分析

实际上，反相 Buck-Boost 在 VIN = |VOUT| 时电感值最大。
```

验证：VIN = |VOUT| 时，D = 0.5

```
L_max = |VOUT|² / (r × IOUT × fSW × 2|VOUT|)
      = |VOUT| / (2 × r × IOUT × fSW)
```

## 3. 四开关非反相 Buck-Boost 电感设计

### 3.1 拓扑结构

```
VIN ---+--[S1]---+--[S2]---+---[L]---+--- VOUT
       |         |         |          |
     [Cin]      GND      [S3]       [C]---[RL]
       |                   |          |
GND ---+-------------------+----------+--- GND

S1, S2 = 高侧 MOSFET（Buck 段）
S3     = 低侧 MOSFET（Boost 段）
S4     = 同步整流（可选，实际常集成在控制器内）

电感在输出侧，类似 Buck
```

这是目前**最主流的商业化升降压方案**（如 TI TPS63xxx 系列、MPS MPQ4430、ADI LTC3130）。

### 3.2 三种工作模式

四开关 Buck-Boost 根据 VIN 与 VOUT 的关系自动切换模式：

| 模式 | 条件 | 工作状态 | 等效拓扑 |
|------|------|---------|---------|
| **Buck 模式** | VIN > VOUT + 裕量 | S1 开关，S2 常通，S3 互补 | Buck |
| **Boost 模式** | VIN < VOUT - 裕量 | S1 常通，S3 开关，S2 互补 | Boost |
| **Buck-Boost 过渡模式** | VIN ≈ VOUT | S1+S3 交替开关 | 4开关 Buck-Boost |

### 3.3 电感值计算 — 关键分析

由于该拓扑在 Buck 模式和 Boost 模式之间切换，**电感值必须同时满足两种模式的需求**：

**Buck 模式要求**：

```
L_buck = VOUT × (VIN(max) - VOUT) / (r_buck × IOUT × fSW × VIN(max))
```

**Boost 模式要求**：

```
L_boost = VIN(min) × (VOUT - VIN(min)) / (r_boost × IIN(max) × fSW × VOUT)
```

其中 IIN(max) = IOUT × VOUT / (VIN(min) × η)

**最终电感值取两者中的较大值**：

```
L = max(L_buck, L_boost)
```

> ⚠️ **核心区别**：四开关 Buck-Boost 电感设计需要**双重校验**，分别按 Buck 和 Boost 模式计算，然后取大值。这是设计中最容易遗漏的步骤。

### 3.4 电感电流应力

**Buck 模式下**（VIN > VOUT）：

```
IL(dc) = IOUT（与 Buck 相同）
IPEAK = IOUT × (1 + r/2)
```

**Boost 模式下**（VIN < VOUT）：

```
IL(dc) = IIN = IOUT × VOUT / (VIN × η)（与 Boost 相同）
IPEAK = IIN × (1 + r/2)
```

**过渡模式**（VIN ≈ VOUT）：

```
IL(dc) ≈ IOUT × (1 + 过渡裕量)
IPEAK = 略高于上述两种模式
```

> **总结**：最大电流应力出现在 **VIN 最低时**（Boost 模式），因为此时 IIN 最大。

### 3.5 过渡模式的特殊问题

当 VIN ≈ VOUT 时，控制器需要在 Buck 和 Boost 模式之间切换，会出现：

- **占空比跳变**：从 D<0.5（Buck）切换到 D>0.5（Boost）
- **电感电流纹波增大**：过渡模式下开关序列更复杂
- **效率凹陷**：4 个 MOSFET 都参与开关，损耗增大
- **电感选型需留裕量**：建议 r 取 0.3~0.4 并检查过渡模式纹波

## 4. SEPIC 变换器电感设计

### 4.1 拓扑结构

```
VIN ---[L1]---+---[SW]---+
               |          |
             [Cac]       GND
               |          |
              [L2]       [D]---+--- VOUT
               |               [C]   [RL]
GND -----------+----------------+-----+

L1 = 输入电感
L2 = 输出电感
CAC = 耦合电容（隔直电容）
```

SEPIC 特点：非反相输出，输入电流连续（由 L1 保证），通过耦合电容 Cac 传递能量。

### 4.2 SEPIC 电感的两种实现方式

#### 方式一：两个独立电感

L1 和 L2 各自独立，可以分别优化。通常取 L1 = L2 以简化设计。

#### 方式二：耦合电感（变压器结构）

L1 和 L2 绕在同一磁芯上，耦合系数 k ≈ 1。

优点：
- 体积更小
- 输入纹波电流可进一步抵消
- 等效漏感 = 谐振电感，可能有用

缺点：
- 需要关注耦合系数和漏感
- 匝比通常为 1:1

### 4.3 电感值计算公式

SEPIC 中 L1 和 L2 的电压应力相同（都等于 VIN + VOUT），工作波形类似。

**L1 的纹波电流**（开关导通时，VIN 加在 L1 上）：

```
ΔIL1 = VIN × D / (L1 × fSW)
```

**L2 的纹波电流**（开关导通时，VOUT 加在 L2 上——极性使电流下降）：

```
ΔIL2 = VOUT × D / (L2 × fSW)
```

若 L1 = L2 = L：

```
ΔIL = ΔIL1 = VIN × D / (L × fSW)（当 L1=L2 时）
```

**占空比**：

```
D = VOUT / (VIN + VOUT)（与反相 Buck-Boost 相同形式）
```

**电感值公式**：

```
        VIN × VOUT
  L = ──────────────────────────
        ΔIL × fSW × (VIN + VOUT)
```

或用纹波系数 r = ΔIL / IIN：

```
        VIN²
  L = ─────────────────────────────────
        r × IIN × fSW × (VIN + VOUT)
```

其中 IIN = IOUT × (VIN + VOUT) / (VIN × η)

### 4.4 各电感电流

| 电流 | L1（输入侧） | L2（输出侧） |
|------|-------------|-------------|
| 直流分量 | IIN = IOUT×(VIN+VOUT)/(VIN×η) | IOUT |
| 纹波 | VIN×D/(L×fSW) | VOUT×D/(L×fSW) |
| 峰值 | IIN×(1+r/2) | IOUT×(1+r/2) |

> **注意**：L1 的直流电流远大于 L2（因为 IIN > IOUT）。L1 的 Isat 要求更高。

### 4.5 耦合电容 Cac 的影响

CAC 的电压 = VIN（平均值），纹波电流较大。其容值影响：

- 容值过小 → 电压纹波大 → 影响电感电流波形
- 容值需满足：CAC ≥ IOUT × D / (fSW × ΔVC)

### 4.6 右半平面零点

SEPIC 与 Boost 类似存在 RHPZ：

```
         RLOAD × (1-D)²
fRHPZ ≈ ─────────────────
         2π × L_eq × D²

其中 L_eq 为等效电感（L1//L2 或耦合后的等效值）
```

## 5. Ćuk 变换器电感设计

### 5.1 拓扑结构

```
VIN ---[L1]---+---[SW]---+---[L2]---+--- VOUT(-)
               |          |           |
              GND        [Cac]       [C]---[RL]
               |          |           |
GND -----------+----------+-----------+--- GND

L1 = 输入电感
L2 = 输出电感
CAC = 能量传递电容
```

Ćuk 特点：反相输出，**输入和输出电流都连续**（两个电感分别滤波），EMI 特性最优。

### 5.2 电感值计算

**L1 纹波**（导通时 VIN 加在 L1 上）：

```
ΔIL1 = VIN × D / (L1 × fSW)
```

**L2 纹波**（导通时 VOUT 反向加在 L2 上）：

```
ΔIL2 = |VOUT| × (1-D) / (L2 × fSW)

注意：Ćuk 的 D = |VOUT| / (VIN + |VOUT|)
```

**电感值公式**（设 L1 = L2 = L）：

```
        VIN × |VOUT|
  L = ──────────────────────────
        ΔIL × fSW × (VIN + |VOUT|)
```

形式上与反相 Buck-Boost 和 SEPIC **完全一致**。

### 5.3 Ćuk 的特殊优势：耦合电感

Ćuk 变换器中 L1 和 L2 **非常适合使用耦合电感**：

- L1 和 L2 的电压波形互补，耦合后纹波电流可以大幅抵消
- 理想情况下（k=1），总输入/输出纹波可趋近于零
- 耦合后的等效电感：M = k√(L1×L2)

使用耦合 Ćuk 电感的优势：
- 体积减少 30%~50%
- 输入输出纹波极小
- 适合 EMI 要求严格的应用

## 7. 电感跨拓扑选型规律总结

通过对 Buck、Boost 和各种升降压拓扑的系统分析，可以提炼出以下核心规律：

### 规律一：电感直流电流 = 该拓扑中"流过电感所在支路"的直流分量

**这是所有拓扑设计的根本出发点。**

| 拓扑 | 电感位置 | IL(dc) | 取决于 |
|------|---------|--------|--------|
| Buck | 输出支路 | **IOUT** | 负载电流 |
| Boost | 输入支路 | **IIN = IOUT/(1-D)** | 负载+升压比 |
| 反相 BB | 能量传递支路 | **IOUT/(1-D)** | 负载+升压比 |
| 4-Sw BB | 输出支路 | **Buck时IOUT / Boost时IIN** | 模式 |
| SEPIC L1 | 输入支路 | **IIN** | 负载+升压比 |
| SEPIC L2 | 输出支路 | **IOUT** | 负载电流 |
| Ćuk L1 | 输入支路 | **IIN** | 负载+升压比 |
| Ćuk L2 | 输出支路 | **IOUT** | 负载电流 |

> **规律**：电感在输入侧 → IL(dc) = IIN（较大）；电感在输出侧 → IL(dc) = IOUT（较小）。
>
> **推论**：相同功率下，**输入侧电感承受更大电流应力**，需要更高的 Isat 和更大的封装。

### 规律二：峰值电流 = IL(dc) × (1 + r/2)，Isat 需求 ∝ IL(dc)

所有拓扑中，峰值电流公式形式统一：

```
IPEAK = IL(dc) × (1 + r/2)
Isat ≥ IPEAK × 1.2 = IL(dc) × (1 + r/2) × 1.2
```

因此：

| 拓扑 | Isat ∝ | 相对应力（VIN=5V, VOUT=12V） |
|------|--------|---------------------------|
| Buck (12V→5V) | IOUT | 1.0× |
| Boost (5V→12V) | IIN = 2.4×IOUT | **2.4×** |
| 反相 BB | IOUT/(1-D) = 3.39×IOUT | **3.4×** |
| SEPIC L1 | IIN = 3.39×IOUT | **3.4×** |
| SEPIC L2 | IOUT | 1.0× |

> **规律**：升降压比越大（VIN 越远离 VOUT），D 越极端，电流应力越大。
>
> **推论**：**高升降压比 = 大电流应力 = 需要更大电感 = 更低效率**。这是所有升降压拓扑的共同瓶颈。

### 规律三：电感值公式的通用形式

所有拓扑的电感值公式都可以统一为：

```
L = V_L × D_eff / (ΔIL × fSW)
```

其中 V_L 是电感两端在开关导通（或关断）阶段的电压，D_eff 是有效占空时间。

| 拓扑 | V_L（导通阶段） | 有效时间 | 等效形式 |
|------|---------------|---------|---------|
| Buck | VIN - VOUT | D×T | L = VOUT(1-D)/(ΔIL×fSW) |
| Boost | VIN | D×T | L = VIN×D/(ΔIL×fSW) |
| 反相 BB | VIN | D×T | L = VIN×D/(ΔIL×fSW) |
| SEPIC L1 | VIN | D×T | L = VIN×D/(ΔIL×fSW) |
| Ćuk L1 | VIN | D×T | L = VIN×D/(ΔIL×fSW) |

> **规律**：导通阶段电感电压越低、占空比越小，所需电感值越小。
>
> **推论**：Buck 模式的电感值需求通常最低（因为 V_L = VIN-VOUT < VIN），所以四开关 Buck-Boost 中 L_buck < L_boost，Boost 模式决定电感下限。

### 规律四：纹波系数 r 的推荐值随拓扑"电感应力等级"递增

| 拓扑 | 推荐 r | 原因 |
|------|--------|------|
| Buck | 0.2~0.3 | 电流应力最低，可用更小 r |
| Boost | 0.3~0.4 | 电流应力中等，需平衡 RHPZ |
| 4-Sw Buck-Boost | 0.3~0.4 | 需同时兼顾 Buck/Boost 两种模式 |
| 反相 Buck-Boost | 0.3~0.4 | 电流应力高，r 不宜过小 |
| SEPIC | 0.3~0.5 | 双电感+耦合电容，复杂度高 |
| Ćuk | 0.3~0.5 | 耦合后可大幅减小有效纹波 |

> **规律**：拓扑越复杂、电流应力越大 → r 推荐值越大 → 电感相对越小 → 补偿 RHPZ 带宽限制。

### 规律五：DCR 损耗随拓扑按 1/(1-D)² 比例放大

| 拓扑 | DCR 损耗 | 相对倍数（D=0.6） |
|------|---------|-----------------|
| Buck | IOUT² × DCR | 1× |
| Boost | IIN² × DCR = [IOUT/(1-D)]² × DCR | **6.25×** |
| 反相 BB | IIN² × DCR | **6.25×** |
| SEPIC L1 | IIN² × DCR | **6.25×** |
| SEPIC L2 | IOUT² × DCR | 1× |

> **规律**：输入侧电感的 DCR 损耗远大于输出侧。升降压拓扑总 DCR 损耗普遍高于 Buck。
>
> **推论**：升降压设计对电感 DCR 要求更严格。大电流场景优先选大封装、低 DCR 型号。

### 规律六：RHPZ 仅存在于含"输出电容→负载→电感"非最小相位路径的拓扑

| 拓扑 | 有 RHPZ？ | RHPZ 频率 | 影响 |
|------|----------|----------|------|
| **Buck** | ❌ 无 | — | 补偿简单 |
| **Boost** | ✅ 有 | RLOAD(1-D)²/(2πLD²) | 限制带宽 |
| **反相 BB** | ✅ 有 | 类似 Boost | 限制带宽 |
| **4-Sw BB** | ⚠️ Boost 模式有 | 类似 Boost | 模式切换需注意 |
| **SEPIC** | ✅ 有 | 类似 Boost（更复杂） | 多极点+RHPZ |
| **Ćuk** | ✅ 有 | 可被耦合电感影响 | 反相输出 |

> **规律**：只要功率传递路径中存在"开关→电容→负载"的间接路径（需经电容传递能量），就会产生 RHPZ。
>
> **推论**：Buck 是唯一无 RHPZ 的基本拓扑，因此补偿最简单、动态响应最快。

### 规律七：电感选型难度排序

综合以上所有因素：

```
电感选型难度：
  Buck < Boost ≈ 4-Sw BB < 反相 BB < SEPIC < Ćuk（独立电感）< Ćuk（耦合电感）
```

| 因素 | 使设计变难的条件 |
|------|----------------|
| 高升压/降压比 | D 趋向极端 → 电流应力↑ |
| 大输出电流 | 所有应力按比例↑ |
| 低输入电压 | IIN 更大 → 电流应力↑ |
| 宽输入范围 | 多工况校验 → 设计变量↑ |
| 高效率要求 | DCR 要求严 → 封装大、成本高 |
| 快速瞬态响应 | RHPZ 限制 → 电感值不能太大 |
| 严格 EMI | 需屏蔽型、SRF 高 → 选择范围窄 |

## 8. 完整选型流程（升降压版）

```
步骤1：确定设计规格
  ├── VIN(min), VIN(max), VIN(typ)
  ├── VOUT
  ├── IOUT(max)
  ├── fSW
  ├── 预估效率 η
  └── 选择拓扑（根据输出极性、效率、成本需求）

步骤2：计算所有工况下的关键参数
  ├── 各 VIN 点的占空比 D
  ├── 各 VIN 点的 IL(dc)
  └── 找出 IL(dc) 最大值（通常在 VIN(min) 时）

步骤3：选择纹波系数 r
  └── 根据拓扑类型和规律四选取

步骤4：计算电感值
  ├── 在 VIN 全范围内扫点
  ├── 对于4-Sw BB：分别按 Buck 和 Boost 模式计算，取 max
  └── 选标准值（向上靠）

步骤5：计算电流额定值
  ├── IPEAK = IL(dc)(max) × (1 + r/2)
  ├── Isat ≥ IPEAK × 1.2
  └── Irms ≥ IL(dc)(max)

步骤6：评估 RHPZ（如有）
  ├── 计算 fRHPZ
  └── 确认 fc < fRHPZ/3

步骤7：评估 DCR 损耗
  ├── PDCR = IL(dc)² × DCR（注意用 IL(dc) 而非 IOUT！）
  └── 确认效率影响可接受

步骤8：检查 SRF 和温度降额

步骤9：对于多电感拓扑（SEPIC/Ćuk）
  ├── 分别校验每个电感的 Isat 和 Irms
  ├── 耦合电感需确认耦合系数 k
  └── 确认耦合电容 Cac 的额定值和容值

步骤10：样机验证
  ├── 温升测试
  ├── 纹波测试
  ├── 效率测试
  ├── 瞬态响应测试
  └── EMI 测试
```

## 9. 实际计算示例

### 示例 1：锂电池 3.0V~4.2V → 3.3V / 2A（4 开关 Buck-Boost）

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 3.0V~4.2V（单节锂电） |
| VOUT | 3.3V |
| IOUT(max) | 2A |
| fSW | 2.5MHz |
| η（预估） | 0.90 |
| r | 0.3 |

**步骤 1：判断工作模式**

- VIN(min) = 3.0V < VOUT = 3.3V → Boost 模式
- VIN(max) = 4.2V > VOUT = 3.3V → Buck 模式
- VIN(typ) = 3.7V > VOUT → Buck 模式
- 需要四开关 Buck-Boost ✅

**步骤 2：Buck 模式计算（VIN = 4.2V）**

```
D = 3.3/4.2 = 0.786
IL(dc) = IOUT = 2A
L_buck = 3.3 × (4.2-3.3) / (0.3 × 2 × 2.5e6 × 4.2)
       = 3.3 × 0.9 / 6300000
       = 2.97 / 6300000
       = 0.471 μH
```

**步骤 3：Boost 模式计算（VIN = 3.0V）**

```
D = (3.3-3.0)/3.3 = 0.091
IIN = 2 × 3.3 / (3.0 × 0.9) = 2.44A
L_boost = 3.0 × 0.091 / (0.3 × 2.44 × 2.5e6)
        = 0.273 / 1830000
        = 0.149 μH
```

**步骤 4：过渡模式检查（VIN ≈ VOUT = 3.3V）**

```
D ≈ 0（理想过渡点）
实际上控制器有过渡区域（±100~200mV），占空比会短暂在 0~1 之间切换
电感纹波在过渡区域可能增大，建议 L ≥ 0.68 μH
```

**步骤 5：选取电感值**

max(L_buck, L_boost) = max(0.471, 0.149) = 0.471 μH

选标准值 **L = 1.0 μH**（留裕量应对过渡模式）

**步骤 6：峰值电流**

Buck 模式（VIN = 4.2V）：

```
ΔIL = 3.3 × (4.2-3.3) / (1.0 × 2.5e6 × 4.2) = 0.283 / 10.5e6... 

重新计算：
ΔIL_buck = VOUT×(VIN-VOUT)/(L×fSW×VIN) = 3.3×0.9/(1.0×2.5e6×4.2)
         = 2.97/(10.5e6) = 0.283A
IPEAK_buck = 2 × (1 + 0.283/(2×2)) = 2 + 0.071 = 2.07A
```

实际 r_buck = 0.283/2 = 0.14（比预设的 0.3 小，因为 L 选了更大值）

Boost 模式（VIN = 3.0V）：

```
ΔIL_boost = VIN×D/(L×fSW) = 3.0×0.091/(1.0×2.5e6) = 0.109A
IPEAK_boost = 2.44 × (1 + 0.109/(2×2.44)) = 2.44 + 0.022 = 2.46A
```

**步骤 7：电流额定值**

```
最大 IPEAK = max(2.07, 2.46) = 2.46A（Boost 模式）
Isat ≥ 2.46 × 1.2 = 2.95A
Irms ≥ max(IOUT, IIN(max)) = max(2, 2.44) = 2.44A
```

**选型结果：**

| 参数 | 要求 | 选择 |
|------|------|------|
| L | 1.0 μH | 1.0 μH |
| Isat | ≥ 2.95A | 3.5A |
| Irms | ≥ 2.44A | 3.0A |
| DCR | ≤ 60mΩ | 40mΩ |
| SRF | > 25MHz | 35MHz+ |

**候选型号**：TDK SPM4020T-1R0M（1.0μH, Isat=5.3A, Irms=3.5A, DCR=40mΩ）

### 示例 2：SEPIC 设计（5V ±2V 输入 → 12V / 0.5A）

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 3V~7V（宽范围输入） |
| VOUT | 12V |
| IOUT | 0.5A |
| fSW | 500kHz |
| η | 0.85 |
| r | 0.4 |
| 拓扑 | SEPIC（非反相、输入电流连续） |

**步骤 1：计算各工况**

VIN = 3V（最大电流应力）：

```
D = 12 / (3+12) = 0.8
IIN = 0.5 × (3+12) / (3 × 0.85) = 7.5 / 2.55 = 2.94A
```

VIN = 5V（典型）：

```
D = 12 / (5+12) = 0.706
IIN = 0.5 × 17 / (5 × 0.85) = 8.5 / 4.25 = 2.0A
```

VIN = 7V：

```
D = 12 / (7+12) = 0.632
IIN = 0.5 × 19 / (7 × 0.85) = 9.5 / 5.95 = 1.60A
```

**步骤 2：电感值计算**

取 L1 = L2 = L，按 VIN = 5V（典型工况）计算：

```
ΔIL1 = VIN × D / (L × fSW) = 5 × 0.706 / (L × 500000) = 7.06/(L × 5e5)
```

要求 ΔIL1 = r × IIN = 0.4 × 2.0 = 0.8A：

```
L = 7.06 / (0.8 × 5e5) = 7.06 / 400000 = 17.65 μH
```

按 VIN = 3V 校验：

```
ΔIL1 = 3 × 0.8 / (17.65 × 5e5) = 2.4 / 8.825e6 = 0.272A
r = 0.272 / 2.94 = 0.093（纹波很小，因为 D 大但 VIN 低）
```

按 VIN = 7V 校验：

```
ΔIL1 = 7 × 0.632 / (17.65 × 5e5) = 4.424 / 8.825e6 = 0.501A
r = 0.501 / 1.60 = 0.313
```

→ L = 17.65 μH 在所有工况下均可接受。选标准值 **22 μH**。

**步骤 3：各电感电流额定值**

L1（输入侧）— VIN = 3V 时最恶劣：

```
IL1(dc) = IIN = 2.94A
ΔIL1(r=0.4, VIN=5V) = 0.8A
IPEAK_L1 = 2.94 × (1 + 0.4/2) = 2.94 × 1.2 = 3.53A
Isat_L1 ≥ 3.53 × 1.2 = 4.24A
Irms_L1 ≥ 2.94A
```

L2（输出侧）：

```
IL2(dc) = IOUT = 0.5A
ΔIL2 = VOUT × D / (L × fSW) = 12 × 0.706 / (22 × 5e5) = 8.47 / 11e6 = 0.77A
IPEAK_L2 = 0.5 + 0.77/2 = 0.885A
Isat_L2 ≥ 0.885 × 1.2 = 1.06A
Irms_L2 ≥ 0.5A
```

**步骤 4：耦合电容 Cac**

```
VCac = VIN(avg) ≈ 5V
ICac_rms ≈ IOUT × √(D × (1-D)) = 0.5 × √(0.706×0.294) = 0.5 × 0.456 = 0.228A
Cac ≥ IOUT × D / (fSW × ΔVc) = 0.5 × 0.706 / (500000 × 0.1) = 7.06 μF
选 Cac = 10μF / 16V（MLCC）
```

**步骤 5：RHPZ**

```
RLOAD = 12 / 0.5 = 24Ω
fRHPZ = 24 × (1-0.706)² / (2π × 22e-6 × 0.706²)
      = 24 × 0.0864 / (2π × 22e-6 × 0.498)
      = 2.074 / 6.89e-5
      = 30.1 kHz
```

→ fc < 10kHz（响应较慢，这是高升压比 SEPIC 的典型限制）

**选型结果：**

| 器件 | 参数 | 选择 |
|------|------|------|
| L1 | 22μH, Isat≥4.24A, Irms≥2.94A | TDK SPM6530T-220M (22μH, Isat=4.5A, Irms=3.0A) |
| L2 | 22μH, Isat≥1.06A, Irms≥0.5A | 可用更小封装，如 SPM4020T-220M |
| Cac | 10μF/16V MLCC | X7R 0805×2 并联 |

### 示例 3：反相 Buck-Boost 生成负电压（5V → -12V / 0.3A）

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 5V |
| VOUT | -12V |
| IOUT | 0.3A |
| fSW | 1MHz |
| η | 0.85 |
| r | 0.4 |

**步骤 1：占空比**

```
D = |-12| / (5 + |-12|) = 12 / 17 = 0.706
```

**步骤 2：电感直流电流**

```
IL(dc) = IOUT / (1-D) = 0.3 / (1-0.706) = 0.3 / 0.294 = 1.02A
```

**步骤 3：电感值**

```
L = VIN² / (r × IOUT × fSW × (VIN + |VOUT|))
  = 25 / (0.4 × 0.3 × 1e6 × 17)
  = 25 / 2040000
  = 12.25 μH
```

选标准值 **15 μH**。

**步骤 4：纹波与峰值电流**

```
ΔIL = VIN × D / (L × fSW) = 5 × 0.706 / (15 × 1e6) = 0.235A
IPEAK = 1.02 + 0.235/2 = 1.14A
Isat ≥ 1.14 × 1.2 = 1.37A
Irms ≥ 1.02A
```

**步骤 5：DCR 损耗**

```
PDCR = 1.02² × DCR
若 DCR = 200mΩ → PDCR = 1.04 × 0.2 = 0.21W（15μH 小封装，DCR 偏大）
若 DCR = 100mΩ → PDCR = 0.10W
```

**选型结果：**

| 参数 | 要求 | 选择 |
|------|------|------|
| L | 15 μH | 15 μH |
| Isat | ≥ 1.37A | 1.8A |
| Irms | ≥ 1.02A | 1.5A |
| DCR | ≤ 150mΩ | 80mΩ |
| SRF | > 10MHz | 15MHz+ |

**候选型号**：TDK SPM4015T-150M（15μH, Isat=1.8A, Irms=1.3A, DCR=130mΩ）

## 10. 选型检查清单

### 通用检查项（所有拓扑）

- [ ] 电感值在 VIN 全范围内扫点验证
- [ ] Isat ≥ IPEAK × 1.2
- [ ] Irms ≥ IL(dc)(max)
- [ ] DCR 损耗可接受（P = IL(dc)² × DCR）
- [ ] SRF > 10 × fSW
- [ ] 高温降额后仍满足
- [ ] 封装满足 PCB 约束

### 拓扑特有检查项

**Buck：**
- [ ] 确认 VIN(min) > VOUT（留裕量）
- [ ] 检查最小负载时的 CCM/DCM 边界

**Boost：**
- [ ] 确认按 IIN 而非 IOUT 选额定值
- [ ] RHPZ 频率可接受

**反相 Buck-Boost：**
- [ ] 确认输出极性需求（负电压）
- [ ] 输出电容需承受 VOUT + VIN 的总电压
- [ ] 反馈回路极性正确

**4-Sw Buck-Boost：**
- [ ] 分别按 Buck 和 Boost 模式校验
- [ ] 过渡模式纹波可接受
- [ ] 控制器支持自动模式切换

**SEPIC：**
- [ ] L1 和 L2 分别校验 Isat 和 Irms
- [ ] 耦合电容 Cac 的额定电压和容值
- [ ] 如用耦合电感，确认耦合系数

**Ćuk：**
- [ ] 同 SEPIC 的双电感校验
- [ ] 耦合电容 Cac 需承受 VIN + |VOUT|
- [ ] 如用耦合电感，关注匝比和耦合系数

## 11. 常见问题与陷阱

### Q1：4-Sw Buck-Boost 和 SEPIC 都可以做升降压，怎么选？

| 对比 | 4-Sw BB | SEPIC |
|------|---------|-------|
| 效率 | **更高**（单电感， fewer losses） | 较低（双电感 + Cac 损耗） |
| 元件数 | 1 电感 + 4 开关 | 2 电感 + 2 开关 + Cac |
| 输入电流 | 脉动（Buck 模式） | **连续**（始终有 L1） |
| 成本 | 控制器复杂（4 开关驱动） | 控制器简单 + 额外电感/电容 |
| EMI | 中 | **更好**（输入电流连续） |
| 功率范围 | 更宽（可达 100W+） | <50W 居多 |

> **建议**：高效率首选 4-Sw BB；需要输入电流连续/低 EMI 选 SEPIC。

### Q2：SEPIC 用两个独立电感还是耦合电感？

| 方案 | 优点 | 缺点 |
|------|------|------|
| 独立电感 | 设计灵活，可分别优化，选择范围大 | 体积大，总成本高 |
| 耦合电感 | 体积小 30%~50%，纹波抵消 | 需要专用型号，关注 k 和漏感 |

> **建议**：空间受限优先耦合电感；设计灵活性优先独立电感。

### Q3：为什么升降压拓扑效率普遍低于 Buck？

1. **电流应力更大**：IL(dc) = IIN > IOUT → 导通损耗更大
2. **更多开关/器件**：4-Sw BB 有 4 个 MOSFET，SEPIC/Ćuk 有额外电容
3. **RHPZ 限制带宽**：环路补偿更保守，可能需要额外损耗元件
4. **过渡模式损耗**：4-Sw BB 在 VIN≈VOUT 时效率凹陷

### Q4：电感在 Buck-Boost 电路中发烫严重怎么办？

排查步骤：
1. **检查直流偏置**：实际 IL(dc) 是否超过 Isat？用示波器看电流波形
2. **检查 DCR 损耗**：I²×DCR 是否过大？测量电感表面温升
3. **检查纹波电流**：是否因为电感值不够导致 ΔIL 过大？
4. **检查频率**：开关频率是否使电感进入 SRF 附近？
5. **检查环境温度**：周围热源是否导致降额不足？

### Q5：如何判断是否需要用升降压拓扑？

```
如果 VIN 范围始终 > VOUT → 用 Buck
如果 VIN 范围始终 < VOUT → 用 Boost
如果 VIN 范围跨越 VOUT → 必须用升降压拓扑
  ├── 需要非反相输出？
  │   ├── 效率优先 → 4-Sw Buck-Boost
  │   ├── EMI 优先 → SEPIC
  │   └── 最低 EMI → Ćuk
  └── 需要反相输出？
      └── 反相 Buck-Boost
```

```

│                                                                         │
│  电感跨拓扑选型 7 大核心规律                                            │
│                                                                         │
│  规律1：IL(dc) 取决于电感所在支路                                       │
│         输入侧 → IIN（大）  输出侧 → IOUT（小）                         │
│                                                                         │
│  规律2：IPEAK = IL(dc) × (1+r/2), Isat ∝ IL(dc)                      │
│         电流应力等级：Boost > Buck（相同功率下）                         │
│                                                                         │
│  规律3：L = V_L × D / (ΔIL × fSW)  统一形式                            │
│         Buck 模式电感需求最小, Boost 模式需求最大                        │
│                                                                         │
│  规律4：r 推荐值随拓扑复杂度递增                                        │
│         Buck:0.2~0.3 → Boost:0.3~0.4 → SEPIC/Ćuk:0.3~0.5              │
│                                                                         │
│  规律5：DCR 损耗 ∝ IL(dc)²                                             │
│         输入侧电感 DCR 损耗远大于输出侧                                 │

│                                                                         │
│  规律6：RHPZ 存在于非直接能量传递拓扑                                   │

│         所有含"电容耦合→输出"路径的拓扑均有 RHPZ                        │
│                                                                         │
│  规律7：设计难度 ∝ 升降压比 × 拓扑复杂度                                │
│         Buck < Boost < 4-Sw BB < 反相 BB < SEPIC < Ćuk               │

│  │ 2. 再算 IPEAK → Isat                         │                      │

│  │ 4. 再验 DCR / SRF / RHPZ                     │                      │
```
