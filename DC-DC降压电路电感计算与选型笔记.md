## 1. Buck 拓扑基础回顾

### 1.1 基本结构

降压（Buck）变换器由以下核心器件组成：

```
VIN ----+--------+
        |        |
      [SW]      [L]----+---- VOUT
        |        |      |
       GND     [D]    [C]    [RL]
        |        |      |      |
GND ----+--------+------+------+-

[SW] = 开关管（MOSFET）    [D] = 续流二极管或同步MOSFET
[L]  = 功率电感            [C] = 输出电容
[RL] = 负载
```

### 1.2 工作原理

Buck 变换器通过开关管周期性导通/关断，配合电感的储能与释能实现降压：

- **开关导通阶段 (ton)**：输入电压加在电感两端，电感电流线性上升，能量储存在电感磁场中
- **开关关断阶段 (toff)**：电感通过续流回路释放能量，电流线性下降，维持负载供电

### 1.3 基本关系

| 参数 | 公式 | 说明 |
|------|------|------|
| 占空比 | D = VOUT / VIN | 理想情况（忽略压降） |
| 开关周期 | TSW = 1 / fSW | fSW 为开关频率 |
| 导通时间 | ton = D × TSW | |
| 关断时间 | toff = (1-D) × TSW | |
| 伏秒积平衡 | VIN × ton = (VIN - VOUT) × toff | 稳态时电感伏秒积为零 |

## 2. 电感值计算公式推导

### 2.1 核心推导

根据电感基本方程：

```
V = L × di/dt
```

在开关导通阶段，电感两端电压为 (VIN - VOUT)：

```
VIN - VOUT = L × ΔIL / ton
```

其中 ton = D / fSW = (VOUT / VIN) / fSW

代入得：

```
VIN - VOUT = L × ΔIL × VIN × fSW / VOUT
```

整理得到 **电感值计算公式**：

```
        VOUT × (VIN - VOUT)
  L = ─────────────────────────
        ΔIL × fSW × VIN
```

### 2.2 用纹波电流系数表示

定义纹波电流系数 r = ΔIL / IOUT，则 ΔIL = r × IOUT：

```
        VOUT × (VIN - VOUT)
  L = ─────────────────────────
        r × IOUT × fSW × VIN
```

也可以写成：

```
        VIN × (1 - D)
  L = ─────────────────
        r × IOUT × fSW
```

> **注意**：当 VIN 最大时，(VIN - VOUT) / (VIN × fSW) 项取最大值，因此 **最坏情况电感值通常在 VIN 最大时计算**。但也有文献指出在占空比 D = 0.5 即 VIN = 2×VOUT 时电感值需求最大。实际应取多个输入电压点计算后取最大值。

### 2.3 纹波电流 ΔIL 的表达式

```
ΔIL = VOUT × (VIN - VOUT) / (L × fSW × VIN)
```

或：

```
ΔIL = (VIN - VOUT) × D / (L × fSW)
```

## 3. 纹波电流系数 r 的选取

### 3.1 什么是 r

纹波电流系数（Ripple Current Ratio）定义为电感纹波电流峰峰值与输出直流电流之比：

```
r = ΔIL / IOUT
```

### 3.2 典型取值范围

| r 值 | 适用场景 | 特点 |
|------|---------|------|
| 0.2 | 低纹波要求、轻载效率优先 | 电感值大，体积大，DCR 可能更大 |
| **0.3** | **通用推荐值（业界标准）** | **平衡体积、效率与纹波** |
| 0.4 | 小体积要求、可接受较大纹波 | 电感值小，体积小，但纹波大 |
| 0.5~1.0 | DCM 边界设计 | 特殊应用 |

### 3.3 r 值对各参数的影响

**r 偏小（电感偏大）：**
- ✅ 输出纹波电压小
- ✅ 峰值电流低，EMI 好
- ✅ 更易保持 CCM（连续导通模式）
- ❌ 电感体积大、成本高
- ❌ DCR 可能偏大（匝数多）
- ❌ 瞬态响应慢（电流爬升慢）

**r 偏大（电感偏小）：**
- ✅ 电感体积小、成本低
- ✅ 瞬态响应快
- ❌ 输出纹波电压大
- ❌ 峰值电流高，需要更大饱和裕量
- ❌ EMI 较差
- ❌ 可能进入 DCM，环路补偿困难

### 3.4 CCM 与 DCM 的边界

当 r = 2 时，电感电流最小值降到零，处于 CCM/DCM 边界。通常设计保证在满载时 r < 2 以维持 CCM 工作。

CCM 条件：

```
IOUT > ΔIL / 2    即    r < 2
```

## 4. 电感电流额定值与饱和电流

### 4.1 电感电流波形

Buck 电路电感电流为三角波叠加直流分量：

```
I(t) = IOUT + ΔIL/2 - (ΔIL/toff)×t    (下降阶段)

峰值电流：IPEAK = IOUT + ΔIL / 2 = IOUT × (1 + r/2)
谷值电流：IVALLEY = IOUT - ΔIL / 2 = IOUT × (1 - r/2)
```

### 4.2 饱和电流 Isat

**定义**：电感值下降到标称值的一定比例（通常为 -10% ~ -30%，不同厂家定义不同）时的电流。

**选型要求**：

```
Isat > IPEAK × 安全裕量

推荐：Isat ≥ IPEAK × 1.2 = IOUT × (1 + r/2) × 1.2
```

以 r = 0.3 为例：

```
Isat ≥ IOUT × (1 + 0.15) × 1.2 = IOUT × 1.38
```

> **经验法则**：饱和电流至少为最大输出电流的 1.3~1.5 倍。

### 4.3 温升电流 / 额定电流 Irms

**定义**：电感在指定温升（通常 40°C）下能承受的有效值（RMS）电流。该电流主要受 DCR 铜损限制。

Buck 电感 RMS 电流近似：

```
IL_rms ≈ IOUT × √(1 + r²/12)
```

以 r = 0.3 为例：

```
IL_rms ≈ IOUT × √(1 + 0.0075) ≈ IOUT × 1.004 ≈ IOUT
```

> 在 r ≤ 0.4 时，RMS 电流非常接近直流输出电流，选型时取 **Irms ≥ IOUT(max)** 即可。

### 4.4 两种电流额定值的关系

| 参数 | 限制因素 | 失效表现 | 选型标准 |
|------|---------|---------|---------|
| Isat（饱和电流） | 磁芯饱和 | 电感骤降，电流尖峰 | ≥ IPEAK × 1.2 |
| Irms（温升电流） | 铜损发热 | 温升过高，损坏 | ≥ IOUT(max) |

> ⚠️ **两者必须同时满足**，缺一不可。数据手册中可能标注为 ISAT / IRMS 或 ISAT / IDC。

## 5. 直流电阻 DCR 的影响

### 5.1 DCR 与效率

电感 DCR 引起的导通损耗：

```
PDCR = IL_rms² × DCR ≈ IOUT² × DCR
```

对效率的影响：

```
η_DCR_loss ≈ 1 - DCR × IOUT / VOUT
```

**举例**：IOUT = 5A, VOUT = 3.3V, DCR = 50mΩ

```
PDCR = 25 × 0.05 = 1.25W
η_loss ≈ 1 - 5×0.05/3.3 = 98.5%
```

### 5.2 DCR 与电感尺寸的矛盾

- 电感值越大 → 匝数越多 → DCR 越大
- 封装越大 → 导体越粗 → DCR 越小
- 需要在电感值、DCR、封装尺寸之间折中

### 5.3 DCR 对负载调整率的影响

在使用电压模式控制或没有电感 DCR 补偿的电流模式控制中，DCR 会引入额外的负载调整误差：

```
ΔVOUT_DCR = IOUT × DCR（在取样点位于电感高侧时）
```

### 5.4 DCR 选型建议

- 在满足电感值和饱和电流的前提下，**DCR 越小越好**
- 通常参考厂家数据手册中的典型/最大 DCR 值
- 注意 DCR 随温度升高而增大（铜线正温度系数约 +3930 ppm/°C）

## 6. 自谐振频率 SRF

### 6.1 什么是 SRF

电感并非理想器件，存在寄生电容（匝间电容、层间电容）。电感与寄生电容形成并联谐振，谐振频率即为 SRF：

```
SRF = 1 / (2π × √(L × Cp))
```

### 6.2 SRF 对工作的影响

| 频率范围 | 电感表现 |
|---------|---------|
| f << SRF | 正常感性，阻抗随频率升高 |
| f ≈ SRF | 阻抗最大（并联谐振） |
| f > SRF | 容性，失去滤波作用 |

### 6.3 SRF 选型要求

```
SRF > 10 × fSW（推荐）
SRF ≥ 5 × fSW（最低要求）
```

**举例**：fSW = 2MHz → SRF 应 > 20MHz

> SRF 过低会导致开关频率处电感阻抗异常，产生振荡和 EMI 问题。

## 7. 电感类型对比

### 7.1 常见磁芯材料

| 类型 | 材料 | 特点 | 适用场景 |
|------|------|------|---------|
| 铁氧体（Ferrite） | MnZn/NiZn 铁氧体 | 高频损耗低，饱和磁通密度中等 | 中高频开关电源（100kHz~5MHz） |
| 合金粉末（Powder） | 铁硅铝/铁镍钼 | 分布气隙，软饱和特性好 | 大电流、需要软饱和的场景 |
| 金属合金（Molding） | 金属磁性粉末 | 高饱和、低剖面、屏蔽好 | 现代集成模块、空间受限应用 |
| 铁氧体屏蔽功率 | 带气隙铁氧体+屏蔽 | 低辐射、高电感 | EMI 要求严格的场景 |

### 7.2 屏蔽 vs 非屏蔽

| 类型 | 优点 | 缺点 |
|------|------|------|
| 屏蔽电感 | 低辐射 EMI，邻近器件影响小 | 成本高，饱和电流可能略低 |
| 非屏蔽电感 | 成本低，饱和电流可能更高 | 辐射 EMI 较大，需布局注意 |

### 7.3 推荐选择

对于大多数现代 Buck 应用（fSW > 500kHz）：
- **首选**：金属合金模压电感（如 TDK SPM、Coilcraft XEL、Vishay IHLM）
- **次选**：屏蔽铁氧体功率电感
- **大电流低频率**：粉末铁芯电感

## 8. 完整选型流程

```
步骤1：确定设计参数
  ├── VIN(min), VIN(max), VIN(nom)
  ├── VOUT
  ├── IOUT(max)
  ├── fSW（开关频率）
  └── 工作环境温度

步骤2：计算所需电感值
  ├── 取 r = 0.2~0.4（推荐 0.3）
  ├── L = VOUT × (VIN(max) - VOUT) / (r × IOUT × fSW × VIN(max))
  └── 在 VIN(min) 处也验算，取较大值

步骤3：计算峰值电流
  ├── ΔIL = r × IOUT(max)
  ├── IPEAK = IOUT(max) + ΔIL/2
  └── IRMS ≈ IOUT(max)（r≤0.4时）

步骤4：确定电流额定值
  ├── Isat ≥ IPEAK × 1.2
  └── Irms ≥ IOUT(max)

步骤5：筛选候选电感
  ├── 电感值 ≥ 计算值（可上靠标准值）
  ├── Isat 满足要求
  ├── Irms 满足要求
  └── 封装尺寸满足 PCB 约束

步骤6：检查 DCR 与效率
  ├── 计算 DCR 损耗：P = IOUT² × DCR
  ├── 评估对效率的影响
  └── 若损耗过大，选更大封装或更低 DCR

步骤7：检查 SRF
  ├── SRF > 10 × fSW
  └── 若不满足，换更小封装或不同系列

步骤8：验证温度升额
  ├── 查数据手册高温降额曲线
  ├── 确认最高工作温度下 Isat 和 Irms 仍满足
  └── 必要时降额使用

步骤9：最终确认
  ├── 确认供应商与交期
  ├── 确认有 SPICE/仿真模型
  └── 制作样机验证温升与纹波
```

## 9. 实际计算示例

### 示例 1：经典中等功率设计

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 12V（范围 8V~16V） |
| VOUT | 3.3V |
| IOUT(max) | 5A |
| fSW | 500kHz |
| r | 0.3 |

**步骤 1：计算电感值**

取 VIN = 16V（最大值）：

```
L = VOUT × (VIN(max) - VOUT) / (r × IOUT × fSW × VIN(max))
  = 3.3 × (16 - 3.3) / (0.3 × 5 × 500000 × 16)
  = 3.3 × 12.7 / (12000000)
  = 41.91 / 12000000
  = 3.49 μH
```

取 VIN = 8V（最小值）验算：

```
L = 3.3 × (8 - 3.3) / (0.3 × 5 × 500000 × 8)
  = 3.3 × 4.7 / 6000000
  = 15.51 / 6000000
  = 2.59 μH
```

→ 选取较大值，**L ≥ 3.49 μH**，选标准值 **4.7 μH**。

**步骤 2：纹波电流**

```
ΔIL = 0.3 × 5 = 1.5A
```

实际纹波（L=4.7μH时，在VIN=16V）：

```
ΔIL_actual = 3.3 × 12.7 / (4.7 × 500000 × 16) = 0.353 × 1.5 ≈ 1.11A
```

实际 r = 1.11 / 5 = 0.22（比预期更低，纹波更小）

**步骤 3：峰值电流与饱和电流**

```
IPEAK = 5 + 1.11/2 = 5.56A
Isat ≥ 5.56 × 1.2 = 6.67A
```

**步骤 4：RMS 电流**

```
IRMS ≈ 5 × √(1 + 0.22²/12) = 5 × 1.002 ≈ 5.01A
Irms ≥ 5A
```

**步骤 5：选型结果**

| 参数 | 要求 | 选择示例 |
|------|------|---------|
| 电感值 | ≥ 4.7 μH | 4.7 μH |
| Isat | ≥ 6.67A | 8A |
| Irms | ≥ 5A | 6A |
| DCR | 尽量小 | ≤ 40mΩ |
| SRF | > 5MHz | 20MHz+ |
| 封装 | 适合 PCB | 5×5mm 或 6×6mm |

**候选型号举例**：
- TDK SPM5030T-4R7M（4.7μH, Isat=7.5A, Irms=6A, DCR=30mΩ）
- Coilcraft XEL5030-472MEC（4.7μH, Isat=8.2A, Irms=6.3A）
- Wurth 74404054470（4.7μH, Isat=7A, Irms=5.5A）

### 示例 2：低压大电流设计（类 CPU 供电）

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 5V（范围 4.5V~5.5V） |
| VOUT | 1.0V |
| IOUT(max) | 20A |
| fSW | 1MHz |
| r | 0.3 |

**步骤 1：电感值**

```
L = 1.0 × (5.5 - 1.0) / (0.3 × 20 × 1000000 × 5.5)
  = 1.0 × 4.5 / (33000000)
  = 0.136 μH ≈ 136 nH
```

选标准值 **150 nH** 或 **220 nH**（略大以减小纹波）。

**步骤 2：峰值电流**

```
ΔIL = 0.3 × 20 = 6A
IPEAK = 20 + 3 = 23A
Isat ≥ 23 × 1.2 = 27.6A
```

**步骤 3：RMS 电流**

```
IRMS ≈ 20A
```

**步骤 4：选型**

需要极低 DCR（<1mΩ）以控制损耗：

```
PDCR = 20² × 1mΩ = 0.4W（可接受）
```

候选型号：
- Vishay IHLP-3212 (220nH, Isat=40A, DCR=0.3mΩ)
- TDK SPM10065T-R15M (150nH, Isat=50A, DCR=0.25mΩ)

### 示例 3：使用 TI TPS54561 的设计

**设计规格（参照 TI TPS54561 数据手册）：**

| 参数 | 值 |
|------|-----|
| VIN | 12V（范围 4.5V~60V） |
| VOUT | 5V |
| IOUT(max) | 5A |
| fSW | 400kHz |
| r | 0.3 |

**电感值计算：**

取 VIN = 60V（最大值）：

```
L = 5 × (60 - 5) / (0.3 × 5 × 400000 × 60)
  = 5 × 55 / 36000000
  = 275 / 36000000
  = 7.64 μH
```

取 VIN = 4.5V（最小值，此时 VOUT=5V 不满足降压条件，实际最低 VIN ≈ 5.5V+Vf）

取 VIN = 12V：

```
L = 5 × (12 - 5) / (0.3 × 5 × 400000 × 12)
  = 35 / 7200000
  = 4.86 μH
```

→ 取 **L = 10 μH**（标准值，兼顾宽输入范围）

**峰值电流：**

```
ΔIL = VOUT × (VIN - VOUT) / (L × fSW × VIN)
    = 5 × (12-5) / (10 × 400000 × 12)
    = 35 / 48000000
    = 0.729A（VIN=12V 时）

IPEAK = 5 + 0.729/2 = 5.36A

VIN=60V 时：
ΔIL = 5×55/(10×400000×60) = 0.458A
IPEAK = 5 + 0.229 = 5.23A
```

**Isat ≥ 5.36 × 1.2 = 6.43A → 选 Isat ≥ 7A**

## 10. 选型检查清单

在最终确定电感之前，逐项确认以下参数：

- [ ] **电感值**：≥ 计算值，在合理范围内（不要偏离超过 2 倍）
- [ ] **饱和电流 Isat**：≥ IPEAK × 1.2（全输入电压范围内校验）
- [ ] **温升电流 Irms**：≥ IOUT(max)（考虑最高环境温度降额）
- [ ] **DCR**：在效率和尺寸允许范围内尽量小
- [ ] **SRF**：≥ 10 × fSW
- [ ] **封装尺寸**：满足 PCB 布局限制
- [ ] **工作温度范围**：覆盖最高环境温度 + 自发热
- [ ] **屏蔽类型**：EMI 敏感应用选屏蔽型
- [ ] **供应商与交期**：确认供货稳定性
- [ ] **仿真模型**：确认有 SPICE 模型或详细参数
- [ ] **安规认证**：如需满足特定标准（如 AEC-Q200 车规）

## 11. 常见问题与陷阱

### Q1：为什么按最大 VIN 计算电感值，而不是最小 VIN？

从公式 L = VOUT × (VIN - VOUT) / (ΔIL × fSW × VIN) 来看，对 VIN 求导可以发现：

```
dL/dVIN = VOUT × VOUT / (ΔIL × fSW × VIN²) > 0（分子为正）
```

实际上这个公式在 VIN = 2×VOUT 时有极值。当 VIN 从 VOUT 增加到 2×VOUT 时 L 增大；VIN > 2×VOUT 后 L 随 VIN 增加而减小。因此：

- 若 VIN(max) < 2×VOUT → L 在 VIN(max) 处最大
- 若 VIN(min) > 2×VOUT → L 在 VIN 接近 2×VOUT 时最大
- 若 VIN 跨越 2×VOUT → L 在 VIN = 2×VOUT 处最大

**建议**：在整个 VIN 范围内扫点计算 L，取最大值。

### Q2：电感值偏大/偏小有哪些后果？

**偏大**：
- 纹波更小，但瞬态响应变慢
- 如果饱和电流不足，可能在负载阶跃时饱和
- 体积和成本增加

**偏小**：
- 瞬态响应快，但纹波大
- 峰值电流高，可能触发过流保护
- 可能进入 DCM 导致环路不稳定
- EMI 恶化

### Q3：为什么饱和电流定义各厂家不同？

- 一些厂家定义为电感下降 10% 时的电流
- 另一些定义为下降 20% 或 30% 时的电流
- 合金粉末电感为"软饱和"，下降是渐进的
- 铁氧体电感为"硬饱和"，下降很陡

**建议**：对比不同型号时，注意同一标准，并在设计中留足裕量。

### Q4：DCR 在不同温度下变化多少？

铜线电阻温度系数约 +3930 ppm/°C：

```
DCR(T) = DCR(25°C) × [1 + 0.00393 × (T - 25)]
```

在 100°C 时：DCR(100°C) = DCR(25°C) × 1.296（增加约 30%）

这意味着高温下铜损显著增加，需要降额使用。

### Q5：多相交错并联时电感如何选择？

多相 Buck 中每相电感值按单相计算，但需注意：
- 每相实际承受的输出纹波电流因交错而减小
- 等效开关频率为 N × fSW（N 为相数）
- 可选用耦合电感（如 Coupled Inductor）进一步缩小体积
- 各相电感应匹配（误差 < 5%），确保均流
