## 1. Boost 拓扑基础回顾

### 1.1 基本结构

升压（Boost）变换器的核心拓扑：

```
VIN ----[L]----+--------+
               |        |
             [SW]      [D]----+---- VOUT
               |        |      |
              GND      [C]    [RL]
               |        |      |
GND -----------+--------+------+

[L]  = 功率电感（输入侧）    [SW] = 开关管（MOSFET）
[D]  = 续流二极管            [C]  = 输出电容
[RL] = 负载
```

> ⚠️ **关键区别**：Boost 电感位于 **输入侧**，电感电流 = 输入电流（非输出电流）。这与 Buck 拓扑（电感在输出侧）有本质不同。

### 1.2 工作原理

- **开关导通阶段 (ton)**：VIN 直接加在电感两端，电感电流从 VIN 经电感、开关管到地线性上升，能量储存在电感中。此时二极管反偏，输出电容独立为负载供电
- **开关关断阶段 (toff)**：电感电压反向，与 VIN 叠加后通过二极管向输出端传递能量，同时为输出电容充电和为负载供电

### 1.3 基本关系

| 参数 | 公式 | 说明 |
|------|------|------|
| 占空比 | D = (VOUT - VIN) / VOUT | 理想情况 |
| 1-D | VIN / VOUT | 关断时间占比 |
| 开关周期 | TSW = 1 / fSW | |
| 导通时间 | ton = D × TSW | |
| 关断时间 | toff = (1-D) × TSW | |
| 输入输出电流关系 | IIN = IOUT / (1-D) | 理想情况（功率守恒） |
| 伏秒积平衡 | VIN × ton = (VOUT - VIN) × toff | 稳态条件 |

### 1.4 关键特性

- **电感电流始终连续**（CCM 模式下）：电感串联在输入回路中，电流不会断流
- **输入电流 = 电感电流**：这是 Boost 的重要特点
- **输出电压恒大于输入电压**（占空比 > 0）
- **右半平面零点（RHPZ）**：Boost 特有的稳定性挑战

## 2. Boost 电感 vs Buck 电感的核心差异

| 对比项 | Buck 电感 | Boost 电感 |
|--------|----------|-----------|
| **位置** | 输出侧 | 输入侧 |
| **直流分量** | IOUT | IIN = IOUT/(1-D) |
| **电感电流含义** | 输出电流+纹波 | 输入电流+纹波 |
| **峰值电流** | IOUT×(1+r/2) | IIN×(1+r/2) = IOUT/(1-D)×(1+r/2) |
| **占空比影响** | D 越大纹波越小 | D 越大，IIN 越大，电感应力越大 |
| **电流应力** | 较低 | 较高（尤其高升压比时） |
| **饱和风险** | 较低 | **高升压比时显著** |
| **右半平面零点** | 无 | 有，限制带宽，影响电感选择 |

> ⚠️ **核心要点**：Boost 电感的直流电流分量 = IIN，而非 IOUT。当升压比（VOUT/VIN）较大时，IIN 可能远大于 IOUT，导致电感承受非常大的电流应力。

## 3. 电感值计算公式推导

### 3.1 核心推导

在开关导通阶段，VIN 直接加在电感两端：

```
VIN = L × ΔIL / ton
```

其中 ton = D / fSW

代入得：

```
VIN = L × ΔIL × fSW / D
```

整理得到 **Boost 电感值计算公式**：

```
        VIN × D
  L = ─────────────────
        ΔIL × fSW
```

将 D = (VOUT - VIN) / VOUT 代入：

```
        VIN × (VOUT - VIN)
  L = ─────────────────────────
        ΔIL × fSW × VOUT
```

### 3.2 用纹波电流系数表示

定义 r = ΔIL / IIN（以 **输入电流** 为基准），则 ΔIL = r × IIN：

```
        VIN × (VOUT - VIN)
  L = ─────────────────────────
        r × IIN × fSW × VOUT
```

由于 IIN = IOUT / (1-D) = IOUT × VOUT / VIN，也可写成：

```
        VIN² × (VOUT - VIN)
  L = ─────────────────────────
        r × IOUT × fSW × VOUT²
```

或简洁形式：

```
        VIN × D
  L = ─────────
        r × IIN × fSW
```

> ⚠️ **注意**：Boost 中 r 的定义基准是 **IIN**（输入电流 = 电感直流电流），不是 IOUT。这是与 Buck 计算最常见的混淆点。

### 3.3 纹波电流 ΔIL 的表达式

```
ΔIL = VIN × D / (L × fSW)
    = VIN × (VOUT - VIN) / (L × fSW × VOUT)
```

### 3.4 最大纹波条件

对 L 公式中关于 VIN 求极值（固定 VOUT）：

```
dL/dVIN = 0  →  VIN = (2/3) × VOUT 时 L 最大
```

即当 VIN = 2/3 × VOUT（占空比 D = 1/3）时，所需电感值最大。

> **设计建议**：如果输入电压范围较宽，应在 VIN = 2/3 × VOUT 附近校验电感值是否足够。当此点不在输入范围内时，取最接近的输入电压点。

## 4. 纹波电流系数 r 的选取

### 4.1 r 的定义

```
r = ΔIL / IIN

其中 IIN = IOUT × VOUT / VIN（理想情况）
       = IOUT / η / (VIN/VOUT)（考虑效率 η 时更保守）
```

### 4.2 典型取值

| r 值 | 适用场景 | 说明 |
|------|---------|------|
| 0.2 | 低纹波、高升压比 | 电感大，但右半平面零点频率低 |
| **0.3~0.4** | **通用推荐** | **平衡电感尺寸与性能** |
| 0.4~0.6 | 空间受限、中等升压比 | 电感小，峰值电流高 |
| 0.8~2.0 | DCM 设计 | 特殊应用（如 LED 驱动） |

### 4.3 Boost 中 r 的特殊考量

**右半平面零点频率与 r 的关系**：

Boost 变换器的右半平面零点（RHPZ）频率：

```
fRHPZ = VOUT × (1-D)² / (2π × L × IIN × D × ... )（简化表达）
```

更精确地：

```
fRHPZ = RLOAD × (1-D)² / (2π × L × D²)

其中 RLOAD = VOUT / IOUT
```

→ **电感值 L 越大，fRHPZ 越低** → 环路带宽必须更低 → 动态响应越差

这是 Boost 设计中 r 取值不能过小的核心原因。

**推荐策略**：

- 低升压比（VOUT/VIN < 2）：r = 0.2~0.3，电感较大可接受
- 中升压比（VOUT/VIN = 2~4）：r = 0.3~0.4
- 高升压比（VOUT/VIN > 4）：r = 0.4~0.6，避免电感过大导致 RHPZ 过低

### 4.4 CCM/DCM 边界

```
边界条件：IIN = ΔIL / 2  →  r = 2

CCM 条件：r < 2  →  IIN > ΔIL / 2
DCM 条件：r > 2  →  IIN < ΔIL / 2
```

> Boost 从 CCM 切换到 DCM 时，传递函数从二阶变为一阶，RHPZ 消失，环路补偿简化。但 DCM 模式器件应力更大。

## 5. 电感电流额定值与饱和电流

### 5.1 电感电流波形

Boost 电感电流（= 输入电流）为三角波叠加直流分量 IIN：

```
I(t) = IIN + 三角纹波

峰值电流：IPEAK = IIN + ΔIL / 2 = IIN × (1 + r/2)
谷值电流：IVALLEY = IIN - ΔIL / 2 = IIN × (1 - r/2)
```

### 5.2 输入电流 IIN 的计算

**理想情况**：

```
IIN = IOUT / (1-D) = IOUT × VOUT / VIN
```

**考虑效率（更保守）**：

```
IIN = IOUT × VOUT / (VIN × η)
```

其中 η 为预估效率（通常 0.85~0.95）。

**举例**：VIN = 3.3V, VOUT = 12V, IOUT = 0.5A, η = 0.9

```
D = (12 - 3.3) / 12 = 0.725
IIN = 0.5 × 12 / (3.3 × 0.9) = 6 / 2.97 = 2.02A
```

> 注意：IIN = 2.02A 远大于 IOUT = 0.5A！这就是 Boost 高升压比时的电流放大效应。

### 5.3 饱和电流 Isat

```
Isat ≥ IPEAK × 安全裕量

推荐：Isat ≥ IPEAK × 1.2 = IIN × (1 + r/2) × 1.2
```

以 r = 0.4 为例：

```
Isat ≥ IIN × 1.2 × 1.2 = IIN × 1.44
```

> **经验法则**：Boost 电感饱和电流至少为最大输入电流的 1.3~1.5 倍。在高升压比设计中，IIN 本身就很大，Isat 要求可能非常苛刻。

### 5.4 温升电流 Irms

Boost 电感 RMS 电流：

```
IL_rms ≈ IIN × √(1 + r²/12)
```

以 r = 0.4 为例：

```
IL_rms ≈ IIN × √(1 + 0.0133) ≈ IIN × 1.007 ≈ IIN
```

→ 选型时取 **Irms ≥ IIN(max)**。

### 5.5 Boost 电流应力总结

| 参数 | 公式 | 说明 |
|------|------|------|
| IIN(dc) | IOUT×VOUT/(VIN×η) | 电感直流分量 |
| ΔIL | r × IIN | 纹波峰峰值 |
| IPEAK | IIN×(1 + r/2) | 峰值电流 |
| IRMS | IIN×√(1+r²/12) ≈ IIN | 有效值电流 |
| Isat 要求 | ≥ IPEAK × 1.2 | 饱和裕量 |
| Irms 要求 | ≥ IIN(max) | 温升裕量 |

## 6. 直流电阻 DCR 的影响

### 6.1 DCR 与效率

Boost 电感 DCR 引起的导通损耗：

```
PDCR = IL_rms² × DCR ≈ IIN² × DCR
```

对效率的影响比 Buck 更显著，因为 IIN > IOUT（高升压比时）：

```
PDCR_Boost / PDCR_Buck ≈ (IIN/IOUT)² = 1/(1-D)²
```

**举例**：VIN = 5V → VOUT = 12V (D ≈ 0.58)

```
IIN = IOUT × 12/(5×0.9) = 2.67 × IOUT
PDCR_Boost = (2.67)² × IOUT² × DCR = 7.11 × IOUT² × DCR
```

> 同样的 DCR，Boost 的铜损是 Buck 的约 7 倍！因此 Boost 对 DCR 的要求更严格。

### 6.2 DCR 与输入电压降

电感 DCR 还会导致有效输入电压降低：

```
VIN_eff = VIN - IIN × DCR
```

在高升压比、大电流场景中，DCR 压降可能显著影响实际占空比：

```
D_actual = (VOUT - VIN_eff) / VOUT > D_ideal
```

→ 实际占空比增大 → 峰值电流进一步增大 → 形成正反馈 → 需要注意热设计。

### 6.3 DCR 选型建议

- Boost 电感 DCR 应尽可能低，尤其在 **低 VIN、高 VOUT、大 IOUT** 的极端条件下
- 大电流 Boost 设计可能需要 DCR < 10mΩ 甚至 < 5mΩ
- 在满足 Isat 和电感值前提下，选择更粗线径/更大封装
- 注意高温下 DCR 增大（铜 +3930ppm/°C）

## 7. 自谐振频率 SRF

### 7.1 与 Buck 相同的原理

SRF 由电感与寄生电容并联谐振决定：

```
SRF = 1 / (2π × √(L × Cp))
```

### 7.2 SRF 选型要求

与 Buck 相同：

```
SRF > 10 × fSW（推荐）
SRF ≥ 5 × fSW（最低要求）
```

### 7.3 Boost 中的额外关注

Boost 电感的寄生电容会通过以下路径影响 EMI：

- 电感寄生电容 → 开关节点 → 二极管 → 输出端
- 在高频时，电感阻抗升高（接近 SRF），开关噪声更容易通过寄生路径耦合

**建议**：
- 选择低寄生电容的电感结构（如多层模压型）
- Boost 应用中 SRF 裕量建议更大（≥ 15×fSW），因高频噪声路径更复杂

## 8. Boost 特有考量：右半平面零点与电感关系

### 8.1 什么是右半平面零点（RHPZ）

Boost 变换器在小信号模型中，存在一个位于复平面右半边的零点，其特点是：

- 增益随频率升高（与左半平面零点相同）
- **相位随频率降低**（与左半平面零点相反！）
- 无法通过补偿网络消除

### 8.2 RHPZ 频率

CCM Boost 的 RHPZ 频率：

```
         RLOAD × (1-D)²
fRHPZ = ─────────────────
         2π × L × D²

其中 RLOAD = VOUT² / POUT = VOUT / IOUT
```

也可写成：

```
         VIN² × VOUT
fRHPZ = ─────────────────
         2π × L × IOUT × (VOUT-VIN)²
```

### 8.3 电感值对 RHPZ 的影响

```
L ↑ → fRHPZ ↓ → 可用环路带宽 ↓ → 动态响应变慢
L ↓ → fRHPZ ↑ → 可用环路带宽 ↑ → 动态响应变快
```

**设计约束**：环路穿越频率 fc 必须满足：

```
fc < fRHPZ / 3 ~ fRHPZ / 5
```

### 8.4 电感选择与动态响应的折中

| 场景 | 电感策略 | 原因 |
|------|---------|------|
| 负载快速变化 | 偏小电感（r 偏大） | fRHPZ 高，响应快 |
| 低输出纹波 | 偏大电感（r 偏小） | 纹波小，但响应慢 |
| 通用设计 | r = 0.3~0.4 | 平衡 |

**举例**：VIN = 5V, VOUT = 12V, IOUT = 1A, L = 10μH

```
D = 7/12 = 0.583
RLOAD = 12/1 = 12Ω
fRHPZ = 12 × (1-0.583)² / (2π × 10e-6 × 0.583²)
      = 12 × 0.1736 / (2π × 10e-6 × 0.34)
      = 2.083 / 2.136e-5
      = 97.5 kHz
```

→ 环路带宽 fc < 97.5kHz / 3 ≈ 32.5kHz

如果 L = 22μH → fRHPZ ≈ 44.3kHz → fc < 14.8kHz（响应明显变慢）

> **结论**：Boost 电感不宜选得过大，否则会严重限制动态响应性能。

## 9. 电感类型与选型建议

### 9.1 推荐的电感类型

| 优先级 | 类型 | 原因 |
|--------|------|------|
| **首选** | 金属合金模压电感 | 高 Isat，低剖面，屏蔽好，DCR 可控 |
| **次选** | 屏蔽铁氧体功率电感 | 高频损耗低，中等 Isat |
| **大电流** | 合金粉末环形电感 | 软饱和特性好，适合高 IIN |
| **低成本** | 非屏蔽铁氧体 | 成本低，但需注意 EMI |

### 9.2 Boost 特殊考量

- **饱和电流要高**：IIN > IOUT 导致 Isat 要求更苛刻，优先选高 Isat 型号
- **屏蔽很重要**：Boost 电感靠近输入端，EMI 辐射容易影响输入源和信号线
- **DCR 要低**：如前分析，Boost 的铜损是 Buck 的 1/(1-D)² 倍
- **考虑温度降额**：高 IIN 导致自发热大，高温下 Isat 可能进一步下降

### 9.3 候选系列举例

| 厂家 | 系列 | 特点 |
|------|------|------|
| TDK | SPM / SPM-A | 金属合金，高 Isat，车规可选 |
| Coilcraft | XEL / XPL | 低 DCR，高 Isat，屏蔽 |
| Vishay | IHLM / IHLP | 模压合金，大电流 |
| Wurth | WE-LQS / WE-Power | 系列全，性价比高 |
| Murata | LQM / DEE | 小型化，多层型 |
| Taiyo Yuden | SRR / MCOIL | 薄型，高饱和 |

## 10. 完整选型流程

```
步骤1：确定设计参数
  ├── VIN(min), VIN(max)
  ├── VOUT
  ├── IOUT(max)
  ├── fSW（开关频率）
  ├── 预估效率 η（初始按 0.85~0.92 估算）
  └── 工作环境温度范围

步骤2：计算输入电流 IIN
  ├── IIN(max) = IOUT(max) × VOUT / (VIN(min) × η)
  └── 这是电感承受的最大直流电流

步骤3：计算占空比
  ├── D(max) = (VOUT - VIN(min)) / VOUT
  └── D(min) = (VOUT - VIN(max)) / VOUT

步骤4：选取纹波系数 r
  ├── 根据升压比和动态响应需求选取 r
  └── 推荐 0.3~0.4（通用），0.4~0.6（高升压比）

步骤5：计算电感值
  ├── 在 VIN 范围内扫点，取最大 L 的点
  ├── L = VIN × D / (r × IIN × fSW)
  ├── 极值点：VIN = 2/3 × VOUT 时 L 最大（若在输入范围内）
  └── 选取标准值（向上靠）

步骤6：计算峰值电流
  ├── ΔIL = r × IIN
  ├── IPEAK = IIN × (1 + r/2)
  └── IRMS ≈ IIN

步骤7：确定电流额定值
  ├── Isat ≥ IPEAK × 1.2
  └── Irms ≥ IIN(max)

步骤8：检查 RHPZ 与环路带宽
  ├── fRHPZ = RLOAD × (1-D)² / (2π × L × D²)
  └── 确认 fc < fRHPZ/3 能满足动态响应需求
      └── 若不满足 → 减小 L → 增大 r → 重新计算

步骤9：检查 DCR 与效率
  ├── PDCR = IIN² × DCR
  ├── 评估对整体效率的影响
  └── 若损耗过大 → 选更大封装/更低 DCR

步骤10：检查 SRF
  └── SRF > 10 × fSW

步骤11：验证温度升额
  ├── 查高温降额曲线
  └── 确认最高温下 Isat、Irms 仍满足

步骤12：最终确认
  ├── 确认封装尺寸
  ├── 确认供应商与交期
  └── 制作样机验证温升与纹波
```

## 11. 实际计算示例

### 示例 1：经典中等升压比设计

**设计规格：**

| 参数 | 值 |
|------|-----|
| VIN | 5V（范围 3.3V~5.5V） |
| VOUT | 12V |
| IOUT(max) | 1A |
| fSW | 1MHz |
| η（预估） | 0.90 |
| r | 0.4 |

**步骤 1：计算 IIN(max)**

在 VIN(min) = 3.3V 时输入电流最大：

```
IIN(max) = 1 × 12 / (3.3 × 0.9) = 12 / 2.97 = 4.04A
```

**步骤 2：计算占空比**

```
D(max) = (12 - 3.3) / 12 = 0.725（VIN = 3.3V 时）
D(min) = (12 - 5.5) / 12 = 0.542（VIN = 5.5V 时）
D(nom) = (12 - 5) / 12 = 0.583（VIN = 5V 时）
```

**步骤 3：计算电感值**

VIN = 3.3V 时（D=0.725, IIN=4.04A）：

```
L = 3.3 × 0.725 / (0.4 × 4.04 × 1e6)
  = 2.3925 / 1616000
  = 1.48 μH
```

VIN = 5V 时（D=0.583, IIN=1×12/(5×0.9)=2.67A）：

```
L = 5 × 0.583 / (0.4 × 2.67 × 1e6)
  = 2.917 / 1068000
  = 2.73 μH
```

VIN = 5.5V 时（D=0.542, IIN=1×12/(5.5×0.9)=2.42A）：

```
L = 5.5 × 0.542 / (0.4 × 2.42 × 1e6)
  = 2.981 / 968000
  = 3.08 μH
```

极值点 VIN = 2/3 × 12 = 8V，不在输入范围内。

→ 最大 L 在 VIN = 5.5V 处，**L ≥ 3.08 μH**，选标准值 **3.3 μH** 或 **4.7 μH**。

> 注意：这个例子中，最大电感值出现在最高 VIN 处（因为 D 较小但 IIN 也小，综合效果）。务必扫点计算！

**步骤 4：峰值电流**

VIN = 3.3V（最大 IIN）：

```
ΔIL = 0.4 × 4.04 = 1.62A
IPEAK = 4.04 + 1.62/2 = 4.85A
```

**步骤 5：电流额定值**

```
Isat ≥ 4.85 × 1.2 = 5.82A
Irms ≥ 4.04A
```

**步骤 6：RHPZ 检查**（在 VIN = 3.3V 时，最恶劣）

```
RLOAD = 12 / 1 = 12Ω
fRHPZ = 12 × (1-0.725)² / (2π × 3.3e-6 × 0.725²)
      = 12 × 0.0756 / (2π × 3.3e-6 × 0.526)
      = 0.9075 / 1.089e-5
      = 83.3 kHz
```

→ 环路带宽 fc < 83.3/3 ≈ 27.8kHz（可接受）

**步骤 7：DCR 损耗**

```
PDCR = 4.04² × DCR
若 DCR = 50mΩ → PDCR = 16.32 × 0.05 = 0.82W
若 DCR = 30mΩ → PDCR = 16.32 × 0.03 = 0.49W
```

**选型结果：**

| 参数 | 要求 | 选择示例 |
|------|------|---------|
| 电感值 | ≥ 3.3 μH | 4.7 μH（留裕量，降低 RHPZ 风险） |
| Isat | ≥ 5.82A | 7A |
| Irms | ≥ 4.04A | 5A |
| DCR | ≤ 50mΩ | 35mΩ |
| SRF | > 10MHz | 15MHz+ |
| 封装 | 适合 | 5×5mm 或 6×6mm |

**候选型号：**
- TDK SPM6530T-4R7M（4.7μH, Isat=7.2A, Irms=5.5A, DCR=35mΩ）
- Coilcraft XEL6030-472MEC（4.7μH, Isat=7.8A, Irms=5.3A, DCR=32mΩ）

> 注意：如果选 4.7μH，RHPZ 会降到约 58.4kHz，fc < 19.5kHz。需确认是否可接受。

| η（预估） | 0.85 |

**步骤 1：IIN(max)**


**设计规格：**

| 参数 | 值 |

| IOUT(max) | 3A |
| fSW | 1.5MHz |
| η（预估） | 0.92 |
| r | 0.3 |

**步骤 1：IIN(max)**

```

D(min) = (5 - 3.6) / 5 = 0.28（VIN = 3.6V）
```

```
  = 1.2 / 2443500
  = 0.491 μH
```

VIN = 3.6V（D=0.28, IIN=3×5/(3.6×0.92)=4.53A）：

```
L = 3.6 × 0.28 / (0.3 × 4.53 × 1.5e6)
  = 1.008 / 2038500
  = 0.494 μH
```

```
D = (5-3.33)/5 = 0.333
```

## 12. 选型检查清单

- [ ] **电感值**：在 VIN 全范围内扫点计算，取最大值，选标准值向上靠
- [ ] **饱和电流 Isat**：≥ IPEAK × 1.2（以 VIN(min) 时 IIN(max) 为基准）
- [ ] **温升电流 Irms**：≥ IIN(max)（考虑效率的保守值）
- [ ] **DCR**：评估 IIN²×DCR 损耗，确保效率和温升可接受
- [ ] **SRF**：≥ 10 × fSW
- [ ] **RHPZ**：fRHPZ 足够高，使得环路带宽能满足动态响应需求
- [ ] **封装尺寸**：满足 PCB 布局约束
- [ ] **工作温度**：高温降额后仍满足 Isat 和 Irms
- [ ] **屏蔽类型**：EMI 敏感应用选屏蔽型
- [ ] **IIN vs IOUT**：确认额定值基准是 IIN（输入电流），不是 IOUT
- [ ] **供应商与交期**：确认供货
- [ ] **样机验证**：实测温升、纹波、效率、动态响应

## 13. 常见问题与陷阱

### Q1：Boost 电感按 IOUT 还是 IIN 选？

**必须按 IIN**。这是最常见的错误！

```
IIN = IOUT × VOUT / (VIN × η)

例：VIN=3.3V, VOUT=12V, IOUT=0.5A → IIN = 0.5×12/(3.3×0.9) = 2.02A
```

IIN 是 IOUT 的 4 倍多！如果按 IOUT=0.5A 选电感，必然饱和。

### Q2：为什么高升压比设计如此困难？

高升压比（VOUT >> VIN）带来三重挑战：

1. **D 接近 1**：IIN = IOUT/(1-D) 急剧增大 → 电流应力巨大
2. **RHPZ 频率低**：D²/(1-D)² 增大 → fRHPZ 降低 → 带宽受限
3. **DCR 损耗大**：IIN²×DCR 随 1/(1-D)² 增长 → 效率恶化

**工程建议**：当升压比 > 4:1 时，考虑：
- 使用变压器隔离拓扑（反激）
- 两级升压
- 耦合电感方案

### Q3：DCM 和 CCM 电感设计有什么区别？

| 对比 | CCM | DCM |
|------|-----|-----|
| 电感值 | 较大（r < 2） | 较小（r > 2） |
| 峰值电流 | 较低 | 较高 |
| Isat 要求 | 较宽松 | 更严格 |
| 环路补偿 | 二阶 + RHPZ | 一阶，无 RHPZ |
| 输出纹波 | 较小 | 较大 |
| 二极管反向恢复 | 有损耗 | 无（自然零电流关断） |
| 适合场景 | 大功率、低纹波 | 小功率、成本敏感 |

### Q4：Boost 电感放在输入侧，EMI 滤波器设计要注意什么？

- 电感输入电流是连续三角波（CCM），含有开关频率谐波
- 需要在 VIN 入口加 LC/π 型 EMI 滤波器
- 电感自身的寄生电容会成为高频噪声的耦合路径
- 选用屏蔽电感可减少辐射 EMI
- PCB 布局上，电感到开关管的高 dv/dt 节点走线要短而粗

### Q5：多相交错 Boost 有什么优势？

多相交错 Boost（2相或更多）的优势：

- **输入纹波电流抵消**：各相交错导通，等效纹波频率 = N×fSW，纹波幅值减小
- **可选更小的输入电容**
- **每相电感值可减小**：总电流分摊到 N 相
- **热分布更均匀**
- 缺点：需要多颗电感和更复杂的控制

## 14. Buck vs Boost 电感对比速查

| 对比项 | Buck | Boost |
|--------|------|-------|
| 电感位置 | 输出侧 | 输入侧 |
| 直流电流 | IOUT | IIN = IOUT/(1-D) |
| 电感值公式 | VOUT×(VIN-VOUT)/(r×IOUT×fSW×VIN) | VIN×D/(r×IIN×fSW) |
| 最大 L 条件 | VIN = 2×VOUT 附近 | VIN = 2/3×VOUT 附近 |
| 典型 r 值 | 0.2~0.4 | 0.3~0.6 |
| IPEAK | IOUT×(1+r/2) | IIN×(1+r/2) |
| Isat 基准 | IOUT 级 | **IIN 级（>>IOUT）** |
| DCR 损耗 | IOUT²×DCR | **IIN²×DCR（>>IOUT²×DCR）** |
| RHPZ | 无 | **有（限制带宽）** |
| EMI 关注 | 输出纹波 | 输入纹波 + 辐射 |
| 设计难度 | 较低 | **较高（尤其高升压比）** |
