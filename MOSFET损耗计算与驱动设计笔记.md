## 1. MOSFET 损耗总览

MOSFET 在开关电源中的总损耗由以下几部分组成：

$$P_{total} = P_{cond} + P_{sw} + P_{gate} + P_{diode}$$

| 损耗类型 | 符号 | 主导因素 | 频率相关性 |
|---|---|---|---|
| 导通损耗 | $P_{cond}$ | $R_{DS(on)}$、占空比、电流 | 无关 |
| 开关损耗 | $P_{sw}$ | $Q_g$、$C_{iss}$、$C_{rss}$、驱动能力 | 正比于 $f_{sw}$ |
| 栅极驱动损耗 | $P_{gate}$ | $Q_g$、驱动电压 | 正比于 $f_{sw}$ |
| 体二极管损耗 | $P_{diode}$ | $V_{SD}$、$t_{rr}$、$Q_{rr}$ | 正比于 $f_{sw}$ |

---

## 2. 导通损耗详解

### 2.1 基本公式

$$P_{cond} = I_{D(rms)}^2 \times R_{DS(on)} \times D$$

其中：
- $I_{D(rms)}$：漏极电流有效值
- $R_{DS(on)}$：导通电阻（受温度影响！）
- $D$：导通占空比

### 2.2 温度对 $R_{DS(on)}$ 的影响

$R_{DS(on)}$ 随结温升高而增大，典型温度系数约为 **+0.4%~0.7%/°C**：

$$R_{DS(on)}(T_j) = R_{DS(on)(25°C)} \times \left[1 + \alpha \times (T_j - 25)\right]$$

> **工程经验**：实际设计时按 $T_j = 100°C$ 计算，此时 $R_{DS(on)}$ 约为 25°C 时的 **1.5~1.7 倍**。

### 2.3 不同拓扑的有效值电流

| 拓扑 | $I_{D(rms)}$ 表达式 | 说明 |
|---|---|---|
| Buck（上管） | $I_{out} \times \sqrt{D}$ | 脉动电流 |
| Buck（下管-同步整流） | $I_{out} \times \sqrt{1-D}$ | 续流电流 |
| Boost（开关管） | $I_{in} \times \sqrt{D}$ | 输入电流脉动 |
| 全桥（每个开关） | $I_{out} / \sqrt{2}$ | 半周期导通 |

### 2.4 关键影响因素

- $R_{DS(on)}$ 随 **$V_{GS}$ 升高而降低**（在阈值以上）
- 低压 MOS（30V 级）通常在 **10V 驱动** 时达到最小 $R_{DS(on)}$
- **逻辑电平 MOS** (Logic Level) 可在 **4.5V** 达到标称 $R_{DS(on)}$
- $R_{DS(on)}$ 典型值分散度大约 ±20%，设计应以 **最大值** 为准

## 3. 开关损耗详解

### 3.1 MOSFET 开关过程

```
VGS
  ^      Miller平台
  |     ┌──────┐
  |    ┌┘      └┐
  |   ┌┘        └┐
  |──┌┘          └────>
  | ┌┘
  |┌┘
  └─────────────────────> t
   t1 t2    t3   t4
   
   t1: 导通延迟 (td(on))   — VGS 从 0 充至 VGS(th)
   t2: 电流上升时间 (tr)    — ID 从 0 升至 Iout
   t3: Miller平台           — VDS 下降，VGS 维持平台电压
   t4: 电压下降完成          — VGS 继续上升至驱动电压
```

### 3.2 开关损耗公式（精确计算）

#### 开通损耗

$$E_{on} = \int_{0}^{t_{sw(on)}} V_{DS}(t) \times I_D(t) \, dt$$

工程近似：

$$E_{on} \approx \frac{1}{2} \times V_{DS(off)} \times I_{D(on)} \times (t_{ri} + t_{fv})$$

$$P_{on} = E_{on} \times f_{sw}$$

#### 关断损耗

$$E_{off} \approx \frac{1}{2} \times V_{DS(off)} \times I_{D(on)} \times (t_{rv} + t_{fi})$$

$$P_{off} = E_{off} \times f_{sw}$$

#### 总开关损耗

$$P_{sw} = P_{on} + P_{off}$$

### 3.3 基于栅极电荷的简化计算

更加实用的方法是利用栅极电荷参数估算开关时间：

| 参数 | 公式 | 说明 |
|---|---|---|
| 开通延迟 | $t_{d(on)} = R_{g(total)} \times C_{iss} \times \ln\left(\frac{V_{drv}}{V_{drv} - V_{th}}\right)$ | $C_{iss} = C_{gs} + C_{gd}$ |
| 电流上升时间 | $t_{ri} = R_{g(total)} \times \frac{Q_{gd}}{V_{drv} - V_{plateau}}$ | Miller 电荷主导 |
| 电压下降时间 | $t_{fv} = R_{g(total)} \times \frac{Q_{gd}}{V_{drv} - V_{plateau}}$ | 实际与 $t_{ri}$ 重叠 |
| 关断延迟 | $t_{d(off)} = R_{g(total)} \times C_{iss} \times \ln\left(\frac{V_{drv}}{V_{plateau}}\right)$ | |
| 关断上升时间 | $t_{rv} = R_{g(total)} \times \frac{Q_{gd}}{V_{plateau}}$ | |
| 关断下降时间 | $t_{fi} = R_{g(total)} \times \frac{Q_{gd}}{V_{plateau}}$ | |

其中：
- $R_{g(total)} = R_{g(driver)} + R_{g(external)} + R_{g(internal)}$
- $V_{plateau}$ 为 Miller 平台电压（典型 3~5V，见数据手册 $V_{gs(pl)}$）
- $Q_{gd}$ 为 Miller 电荷（数据手册中常标为 $Q_{gd}$ 或 $Q_{sw}$）
- $V_{drv}$ 为驱动电压（通常 10V 或 4.5V/5V）

#### 简化工程估算

$$P_{sw} \approx \frac{1}{2} \times V_{DS} \times I_D \times f_{sw} \times (t_{ri} + t_{fi})$$

或进一步利用 $Q_{gd}$：

$$P_{sw} \approx V_{DS} \times I_D \times f_{sw} \times \frac{2 \times Q_{gd} \times R_{g(total)}}{V_{drv} - V_{plateau}} \quad \text{(硬开关)}$$

### 3.4 影响开关损耗的关键因素

| 因素 | 影响 | 设计权衡 |
|---|---|---|
| **$Q_g$ (总栅极电荷)** | 越大开关越慢 | 低 $Q_g$ 好，但 $R_{DS(on)}$ 通常更大 |
| **$Q_{gd}$ (Miller 电荷)** | 直接影响开关时间 | 低压 MOS 通常 $Q_{gd}$ 较小 |
| **$R_g$ (栅极电阻)** | 控制开关速度 | 小 $R_g$ 开关快但 EMI 差 |
| **$C_{rss}$ ($=C_{gd}$)** | 反馈电容，Miller 效应 | 越小开关越快 |
| **驱动电压** | 高 $V_{GS}$ 开关更快 | 受 $V_{GS(max)}$ 限制 |
| **工作频率** | 开关损耗正比于 $f_{sw}$ | 高频需用低 $Q_g$ 管 |

### 3.5 软开关与硬开关

| 开关方式 | 损耗特征 | 典型应用 |
|---|---|---|
| **硬开关** | $P_{sw}$ 显著，$V_{DS}$ 与 $I_D$ 重叠 | 中小功率 Buck/Boost |
| **ZVS (零电压开关)** | 开通损耗几乎为零 | LLC 谐振、相移全桥 |
| **ZCS (零电流开关)** | 关断损耗几乎为零 | 某些谐振变换器 |

> **工程经验**：当 $f_{sw} > 100kHz$ 且为硬开关时，开关损耗通常成为主导损耗，此时应优先选择 $Q_g$ 和 $Q_{gd}$ 小的 MOSFET。

## 4. 栅极驱动损耗

### 4.1 公式

$$P_{gate} = Q_g \times V_{drv} \times f_{sw}$$

其中：
- $Q_g$：总栅极电荷（数据手册典型值）
- $V_{drv}$：驱动电压摆幅（$V_{GS(on)} - V_{GS(off)}$）
- $f_{sw}$：开关频率

### 4.2 驱动芯片功耗

驱动芯片本身也有损耗：

$$P_{driver} = Q_g \times V_{drv} \times f_{sw} + P_{driver(quiescent)}$$

> **注意**：虽然栅极驱动损耗通常较小（数十 mW），但当频率极高（>500kHz）或驱动多个并联 MOSFET 时不可忽略。

---

## 5. 体二极管损耗

### 5.1 导通损耗

同步整流中，下管体二极管在死区时间内导通：

$$P_{diode(cond)} = V_{SD} \times I_{out} \times t_{dead} \times f_{sw}$$

### 5.2 反向恢复损耗

$$P_{rr} = Q_{rr} \times V_{DS} \times f_{sw}$$

> **关键**：体二极管反向恢复 $Q_{rr}$ 在高频硬开关下会产生显著损耗，这是 Si MOSFET 的固有问题。SiC 和 GaN 器件的 $Q_{rr}$ 极小，在高频应用中优势明显。

## 6. 驱动电路设计流程

### 6.1 设计步骤总览

```
1️⃣ 确定开关频率 fsw
       ↓
2️⃣ 确定驱动电压 Vdrv（5V / 10V / 12V）
       ↓
3️⃣ 根据 ID、VDS 初选 MOSFET
       ↓
4️⃣ 提取关键参数：Qg, Qgd, RDS(on), Ciss, Crss
       ↓
5️⃣ 计算损耗：Pcond + Psw + Pgate + Pdiode
       ↓
6️⃣ 判断总损耗是否 ≤ PD(max)
       ↓ 否 ↑
7️⃣ 确定驱动电流能力
       ↓
8️⃣ 计算外部栅极电阻 Rg(ext)
       ↓
9️⃣ 评估开关速度与 EMI
       ↓
🔟 验证热设计（结温 < 125°C）
```

### 6.2 驱动电流计算

驱动芯片所需峰值电流：

$$I_{gate(peak)} = \frac{V_{drv} - V_{plateau}}{R_{g(total)}}$$

或利用 $Q_g$ 估算平均驱动电流：

$$I_{gate(avg)} = Q_g \times f_{sw}$$

驱动芯片选择依据：

| 应用 | 推荐驱动电流 | 说明 |
|---|---|---|
| 小信号 MOS（SOT-23） | 0.1~0.5A | SI2302 这类 |
| 中小功率 MOS（DFN 5×6） | 1~2A | 多数 DC-DC 应用 |
| 大功率 MOS（TO-220/263） | 2~4A | IRF3205 等 |
| 并联 MOS / IGBT | 4~9A | 大电流应用 |
| SiC / GaN | 5~10A+ | 高频、快速开关 |

### 6.3 驱动拓扑选择

| 拓扑 | 优点 | 缺点 | 适合场景 |
|---|---|---|---|
| **简单推挽** | 结构简单，成本低 | 关断速度受限 | 低频、不敏感应用 |
| **栅极串电阻** | 可独立控制充放电速度 | 增加开关损耗 | 通用 |
| **关断加速二极管**（D+并联Rg） | 开通慢（EMI好），关断快（损耗小） | 多一个二极管 | 优化设计常用 |
| **互补驱动（PNP+NPN）** | 低阻抗关断，抗米勒效应 | 元件多 | 大功率 |
| **专用驱动 IC** | 集成功能，可靠 | 成本略高 | 推荐方案 |

## 7. 驱动电阻选型

### 7.1 $R_{g(ext)}$ 的作用

- 控制开关速度（dv/dt、di/dt）
- 抑制栅极振荡（与 $C_{iss}$ 和 PCB 寄生电感形成 LC 振荡）
- 调节 EMI
- 限制驱动 IC 的峰值电流

### 7.2 典型值参考

| 应用场景 | $R_{g(ext)}$ 典型值 | 说明 |
|---|---|---|
| 高频 DC-DC（>300kHz） | 1~5Ω | 追求效率 |
| 中频功率转换（100~300kHz） | 5~15Ω | 效率与 EMI 平衡 |
| 低频大功率（<100kHz） | 10~33Ω | 兼顾 EMI |
| SiC MOSFET | 2.5~10Ω | 需抑制高频振荡 |
| 关断加速路径 | 0~2Ω | 常串一个二极管 |

### 7.3 栅极振荡抑制

栅极回路谐振频率：

$$f_{osc} = \frac{1}{2\pi \sqrt{L_{loop} \times C_{iss}}}$$

- $L_{loop}$ 为栅极驱动回路寄生电感
- 为抑制振荡，需满足：
  $$R_{g(total)} > 2 \sqrt{\frac{L_{loop}}{C_{iss}}}$$

### 7.4 实用推荐电路

```
            ┌──────────────┐
            │  驱动 IC      │
            │  OUT          ├────┬──────────┐
            └──────────────┘    │          │
                                │          │
                               ┌┴┐         │
                               │Rg(on)    ├┘
                               └┬┘        D│
                                │         ──┤
                                │          │
                                ├──────────┘
                                │
                               ┌┴┐
                               │Rg(off)    ← 可选（与D并联）
                               └┬┘
                                │
                            ┌───┴───┐
                            │ 栅极   │
                            └───────┘
```

- **Rg(on)** 控制开通速度
- **D + Rg(off)** 控制关断速度，可独立调节
- 通常可先取 $R_{g(on)} = R_{g(off)}$，再根据实际波形优化

## 8. 实际案例计算

### 8.1 案例：12V→5V Buck 转换器

**设计参数：**
- $V_{in} = 12V$, $V_{out} = 5V$, $I_{out} = 10A$
- $f_{sw} = 200kHz$
- 选择 **BSC014N04LS** (40V, 1.4mΩ, TDSON-8FL)

**MOSFET 关键参数：**
- $V_{DS(max)} = 40V$
- $R_{DS(on),max} = 1.4mΩ @ V_{GS}=10V$
- $Q_g = 61nC @ 10V$
- $Q_{gd} \approx 9.8nC$（典型）
- $C_{iss} = 4200pF$
- $V_{plateau} \approx 3.5V$

**Step 1：占空比**

$$D = V_{out} / V_{in} = 5/12 = 0.417$$

**Step 2：导通损耗（上管）**

$$I_{D(rms)} = I_{out} \times \sqrt{D} = 10 \times \sqrt{0.417} = 6.46A$$

$$T_j 估算 100°C: R_{DS(on)} \approx 1.4 \times 1.6 = 2.24mΩ$$

$$P_{cond} = 6.46^2 \times 0.00224 \times 0.417 = 0.039W \quad (\text{非常小!})$$

**Step 3：开关损耗（硬开关估算）**

取 $R_{g(total)} = 5Ω$（驱动 IC 内部 1Ω + 外部 4Ω）

$$t_{ri} \approx R_{g(total)} \times \frac{Q_{gd}}{V_{drv} - V_{plateau}} = 5 \times \frac{9.8nC}{10 - 3.5} = 7.5ns$$

$$t_{fi} 估算 = 5 \times \frac{9.8nC}{3.5} = 14ns$$

$$P_{sw} = \frac{1}{2} \times 12 \times 10 \times 200k \times (7.5 + 14) \times 10^{-9} = 0.258W$$

**Step 4：栅极驱动损耗**

$$P_{gate} = 61nC \times 10V \times 200kHz = 0.122W$$

**Step 5：总损耗**

$$P_{total} \approx 0.039 + 0.258 + 0.122 \approx 0.42W$$

**Step 6：结温估算**

查数据手册 $R_{thJA} = 50K/W$（FR4 板上）

$$T_j \approx 25°C + 0.42 \times 50 = 46°C \quad \text{远低于 125°C，设计裕量充足}$$

> 这个例子说明 BSC014N04LS 在 200kHz 下有极大的裕量，可以尝试提高频率（减小电感尺寸）或降低栅极电阻（进一步提高效率）。

### 8.2 案例对比：不同 MOSFET 对损耗的影响

| 参数 | BSC014N04LS (40V) | IRF3205 (55V) | SI2302 (20V) |
|---|---|---|---|
| $R_{DS(on)}$ | 1.4mΩ | 8mΩ | 85mΩ |
| $Q_g$ | 61nC | 78nC | 10nC |
| $Q_{gd}$ | 9.8nC | 34nC | ~2nC |
| $P_{cond}$ (10A Buck) | 0.039W | 0.22W | 太高不适用 |
| $P_{sw}$ (200kHz) | 0.258W | 0.66W | — |
| 适用场景 | **大电流功率级** | 中电流功率级 | **信号/小功率** |

## 9. 工程经验与注意事项

### 9.1 设计核心理念

```
低 RDS(on)  ←→ 高 Qg（大尺寸die）
低 Qg       ←→ 高 RDS(on)（小尺寸die）
```

**这就是基本矛盾**：追求低导通损耗必然导致高开关损耗，反之亦然。

**选择策略：**
- **低频（<100kHz）**：选 $R_{DS(on)}$ 尽可能小的管，牺牲 $Q_g$
- **高频（>300kHz）**：选 $Q_g$ 小的管，接受稍高的 $R_{DS(on)}$
- **中频（100~300kHz）**：优值系数 $FOM = R_{DS(on)} \times Q_g$ 越小越好

### 9.2 数据手册阅读要点

| 参数 | 单位 | 注意点 |
|---|---|---|
| $R_{DS(on)}$ | mΩ | 分清楚 **Typ** 和 **Max**，设计用 **Max** |
| $V_{GS(th)}$ | V | 有 Min 和 Max，驱动电压需远高于 Max |
| $Q_g$ | nC | 与测试条件相关（$V_{GS}$ 摆幅、$V_{DS}$） |
| $Q_{gd}$ | nC | **Miller 电荷**，开关损耗的关键参数 |
| $I_D$ | A | 有 $T_C=25°C$ 和 $T_C=100°C$，差距很大 |
| $E_{AS}$ | mJ | 雪崩能量，用于判断是否需加吸收电路 |
| $R_{thJC}$ | °C/W | 结到壳的热阻，散热设计依据 |

### 9.3 常见问题

**Q：为什么手册里的 $Q_g$ 和我算的不一样？**
- $Q_g$ 与 $V_{DS}$ 测试条件有关，$V_{DS}$ 越高 $Q_{gd}$ 越大
- 应选择最接近实际应用的测试条件

**Q：栅极波形有振荡怎么办？**
- 增大 $R_{g(ext)}$ 是最简单的办法
- 优化栅极驱动回路的 PCB 布局（减小环路面积）
- 在栅极加一个小磁珠（如 60Ω@100MHz）

**Q：什么时候需要负压关断？**
- 当 $dv/dt$ 很大（>10V/ns），米勒效应可能导致自导通
- 桥式电路的上管关断时，下管栅极可能被米勒电流抬高
- 解决方案：使用负压关断（$V_{GS(off)} = -3V$ 到 -5V）

**Q：并联 MOSFET 需要注意什么？**
- 每个 MOSFET 栅极需单独串电阻（防止振荡）
- 尽量用同一批次、同一型号的管子
- PCB 布局要对称
- 总的 $Q_g$ 累加，驱动能力要够

### 9.4 实用的 MOSFET 优值系数 (FOM)

| FOM | 定义 | 评价维度 |
|---|---|---|
| $R_{DS(on)} \times Q_g$ | — | 综合性能，越小越好 |
| $R_{DS(on)} \times C_{oss}$ | — | 同步整流优值 |
| $R_{DS(on)} \times Q_{gd}$ | — | 开关速度优值 |

> **建议**：选型时把几个候选管子的 $R_{DS(on)} \times Q_g$ 算一下，通常 FOM 最小的就是最优选择。
