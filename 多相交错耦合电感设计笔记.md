# 多相交错耦合电感设计笔记

> 来源：Feiyang Zhu 博士论文《Investigation of Multiphase Coupled Inductor Topologies for Point-of-Load Applications》（VT, 2023）
> 适用场景：多相 Buck 变换器给 CPU/GPU 供电（数据中心 + 移动设备）

---

## 一、为什么要用耦合电感？

### 普通电感的两难处境

多相 Buck 用的普通电感有个**矛盾**：

```
想要稳态纹波小  →  需要大电感 L  ↗
                              ↖  想要动态响应快  →  需要小电感 L
                              ⬇
                        怎么选都吃亏！
```

- **L 太大** → 负载电流突变时，电感电流跟不上，输出电压跌得一塌糊涂
- **L 太小** → 稳态纹波大，MOSFET 和电感的损耗都高

### 耦合电感怎么破的

耦合电感把多个相的电感做到一个磁芯里，让磁路互相影响。结果就是：

```
同一个电感器，有两种等效电感：

  Lss（稳态电感） → 大！  控制稳态纹波小
  Ltr（暂态电感） → 小！  让动态响应快

两不耽误，鱼和熊掌兼得。
```

关键指标：**Lss / Ltr 的比值越大越好**。耦合越强、相数越多，这个比值就越大。

## 二、三种耦合电感结构

根据耦合方式分为三大类，各有各的脾气：

### 1. DCL — 直接耦合电感（Direct-Coupled Inductor）

**怎么工作的：**
多个电感绕在同一个多腿磁芯上，相与相之间通过磁芯里的磁路直接耦合。

```
   ┌──┐  ┌──┐  ┌──┐  ┌──┐
   │L1│  │L2│  │L3│  │L4│
   └─┬┘  └─┬┘  └─┬┘  └─┬┘
     └────┴────┴────┘
       共用磁芯
```

| 优点 | 缺点 |
|---|---|
| ✅ 相数少时（2~4相）体积最小 | ❌ 相数多了磁芯结构极其复杂 |
| ✅ 有磁通对消效果，磁芯利用率高 | ❌ 耦合不对称 → 各相电流纹波不一样 |
| | ❌ 绕组走线长 → DCR 大 → 铜损大 |

**适合：** 相数少（2~4相）的场景

### 2. ICL — 间接耦合电感（Indirect-Coupled Inductor）

**怎么工作的：**
每一相电感都有自己的独立磁芯。每个磁芯上除了主绕组，还多绕了一个附加绕组。把所有附加绕组串起来形成一个**耦合绕组环**，靠这个环里的电流来实现耦合。

```
  相1         相2         相3         相4
 ┌───┐      ┌───┐      ┌───┐      ┌───┐
 │ L │      │ L │      │ L │      │ L │
 │ + │      │ + │      │ + │      │ + │
 └─┬─┘      └─┬─┘      └─┬─┘      └─┬─┘
   └──────────┴──────────┴──────────┴────  ← 耦合绕组环 (加一个 Lc 控制耦合度)
```

| 优点 | 缺点 |
|---|---|
| ✅ 结构简单，每相一模一样→ **天然对称耦合** | ❌ 需要多一个 Lc（额外电感）来控制耦合度 |
| ✅ 相数随便加，扩展性极好 | ❌ 没有磁通对消 → 磁芯体积比 DCL 大 |
| ✅ 设计方便，每个磁芯独立 | |

**适合：** 相数多的场景（4相以上），目前工业界很流行

### 3. HCL — 混合耦合电感（Hybrid-Coupled Inductor）

**怎么工作的：**
先两两一组做 DCL（直接耦合，利用磁通对消），再把每组通过 ICL 的方式（耦合绕组环）连起来。DCL 和 ICL 的混血儿。

```
  ┌──────┐      ┌──────┐      ┌──────┐
  │ L1 L2│      │ L3 L4│      │ L5 L6│
  │ EI磁芯│      │ EI磁芯│      │ EI磁芯│
  └──┬───┘      └──┬───┘      └──┬───┘
     └─────────────┴─────────────┴──  ← 耦合绕组环
```

| 优点 | 缺点 |
|---|---|
| ✅ 有 DCL 的磁通对消 → 磁芯比 ICL 小 | ❌ 也需要额外电感 Lc |
| ✅ 有 ICL 的扩展性和对称性 | ❌ 两种耦合机理共存，设计更复杂 |
| ✅ 综合性能最好 | |

**适合：** 要求高性能的大电流场景（8相以上）

### 怎么选？一张表搞定

| 场景 | 推荐结构 | 原因 |
|---|---|---|
| 2~4相，体积优先 | **DCL** | 相数少时 DCL 体积最小 |
| 4~8相，灵活优先 | **ICL** | 结构简单，对称性好，扩展方便 |
| 8相以上，既要小又要好 | **HCL** | 融合两者优点，磁芯利用率高 × 扩展性好 |

## 三、两个核心参数：Lss 和 Ltr

这是设计耦合电感首先要搞明白的两个数：

### Lss（稳态电感）

**定义：** 正常工作时，每个开关周期内电感电流的等效电感。

**作用：** 决定稳态电流纹波大小。

**公式（两相负耦合）：**

$$L_{ss} = L_{self} \times \frac{1 + \alpha}{1 - \alpha^2}$$

其中 $\alpha$ 是耦合系数（负耦合取正数）。

### Ltr（暂态电感）

**定义：** 负载突变时，电感电流变化的等效电感。

**作用：** 决定动态响应的快慢。

**公式（两相负耦合）：**

$$L_{tr} = L_{self} \times \frac{1 - \alpha}{1 - \alpha^2} = \frac{L_{self}}{1 + \alpha}$$

### 设计目标

```
✅  Lss 要大  →  稳态纹波小 →  损耗小
✅  Ltr 要小  →  动态响应快 →  电压尖峰小

关键指标：Lss/Ltr 的比值
  普通电感：   1
  两相耦合：   1~3（看耦合系数）
  四相耦合：   1~5
  八相耦合：   1~7
```

### 算个例子

假设自感 $L_{self} = 100nH$，耦合系数 $\alpha = 0.7$：

$$L_{ss} = 100 \times \frac{1 + 0.7}{1 - 0.7^2} = 100 \times \frac{1.7}{0.51} \approx 333nH$$

$$L_{tr} = \frac{100}{1 + 0.7} \approx 59nH$$

$$L_{ss} / L_{tr} \approx 5.6$$

相当于用一个 100nH 的电感器，实现了 333nH 的稳态效果和 59nH 的动态效果！

## 四、ICL 设计实战步骤（4 相为例）

### 第 1 步：确定设计目标

```yaml
输入电压:      12V
输出电压:      1.8V
最大负载电流:  200A
相数:         4相
开关频率:     500kHz
输出纹波要求:  ≤10mV
电压尖峰要求:  ≤50mV (0-200A 跳变)
```

### 第 2 步：算 Lss 和 Ltr

先算需要的稳态电感和暂态电感：

- **Lss**：根据纹波要求反推

$$L_{ss} = \frac{V_{in} - V_{out}}{2 \times \Delta I_{ripple} \times f_{sw}} \times D$$

- **Ltr**：根据动态响应要求反推

$$L_{tr} = \frac{V_{out} \times t_{response}}{\Delta I_{load}}$$

### 第 3 步：确定需要的耦合系数

对于 ICL，Lss 和 Ltr 的关系是：

$$\frac{L_{ss}}{L_{tr}} = \frac{n(1 - D) + \alpha[1 + (n - 1)D]}{n(1 - D) - \alpha[1 + (n - 1)D]}$$

其中 n 是相数，D 是占空比。

有这个比值就能反推出需要的耦合系数 $\alpha$。

### 第 4 步：确定自感 Lself 和额外电感 Lc

ICL 的核心设计公式：

$$L_{self} = L_{tr} \times (1 + \alpha)$$

$$L_c = \frac{L_{self} \times (1 + \alpha)}{n \times \alpha} \times N_{aux}^2$$

其中 $N_{aux}$ 是附加绕组与主绕组的匝比。

### 第 5 步：磁芯设计

每一相的磁芯独立设计，跟做普通电感一样：

1. 选磁芯材料（铁氧体、金属粉芯等）
2. 根据 $L_{self}$ 和峰值电流算匝数
3. 验算磁通密度（不要饱和）
4. 算损耗（铜损 + 铁损）

### 第 6 步：验证与调试

打样测试，主要看三个东西：

```
1. 稳态电流纹波 → 验证 Lss 对不对
2. 动态电压尖峰 → 验证 Ltr 对不对  
3. 效率 → 验证损耗算没算对
```

## 五、避坑指南

### 坑 1：ICL/HCL 的 Lss 和 Ltr 会打架

**现象：** 想减小 Ltr 提升动态，结果 Lss 也跟着小了，纹波变大。

**原因：** ICL/HCL 的 Lss 和 Ltr 是通过同一个耦合系数 α 绑定的，动一个另一个也跟着动。

**解决办法（论文第三章）：**
用**变耦合系数**——在耦合绕组环里放一个**阶梯气隙电感**，大电流时自动减小 Lc，从而在动态时降低耦合、获得更小的 Ltr。

```
正常工作时：Lc 大 → 耦合强 → Lss 大 → 纹波小
负载跳变时：Lc 小 → 耦合弱 → Ltr 小 → 响应快
```

### 坑 2：DCL 相数多了耦合不对称

**现象：** 中间相和边相的磁路长度不一样，导致耦合系数不一样，各相电流纹波不平衡。

**解决办法：** 相数多就别用 DCL，换 ICL 或 HCL。

### 坑 3：ICL/HCL 的额外电感 Lc 占地方

**现象：** 为了调耦合度需要多加一个电感 Lc，白白占用 PCB 面积。

**解决办法（论文第四章）：**
通过改进磁芯结构（比如五腿磁芯），把 Lc 的功能集成到主磁芯里，省掉那个额外电感。

### 坑 4：PCB 内嵌电感做不高耦合

**现象：** 想做低高度（<1mm）的 PCB 内嵌耦合电感，但耦合系数上不去。

**解决办法（论文第五章）：**
用**非对称串联绕线方式**，通过调整气隙位置和过孔分布来优化耦合。

## 六、总结：设计流程速查

```
┌──────────────────────────────────────────┐
│ 1. 明确需求                              │
│    - 输入/输出电压、电流、频率、相数      │
│    - 纹波要求、动态响应要求               │
└────────────────┬─────────────────────────┘
                 ▼
┌──────────────────────────────────────────┐
│ 2. 算 Lss 和 Ltr                         │
│    - 纹波要求 → Lss                      │
│    - 动态要求 → Ltr                      │
└────────────────┬─────────────────────────┘
                 ▼
┌──────────────────────────────────────────┐
│ 3. 选结构类型                            │
│    ≤4相 → DCL（体积优先）                │
│    4~8相 → ICL（灵活优先）               │
│    ≥8相 → HCL（性能优先）                │
└────────────────┬─────────────────────────┘
                 ▼
┌──────────────────────────────────────────┐
│ 4. 算关键参数                            │
│    - 需要的耦合系数 α                    │
│    - 自感 Lself                          │
│    - 额外电感 Lc（ICL/HCL）              │
└────────────────┬─────────────────────────┘
                 ▼
┌──────────────────────────────────────────┐
│ 5. 磁芯设计                              │
│    - 选材料                              │
│    - 算匝数                              │
│    - 验饱和                              │
│    - 算损耗                              │
└────────────────┬─────────────────────────┘
                 ▼
┌──────────────────────────────────────────┐
│ 6. 打样测试                              │
│    - 测稳态纹波 → 验证 Lss              │
│    - 测动态响应 → 验证 Ltr              │
│    - 测效率                              │
└──────────────────────────────────────────┘
```

## 七、一些有用的经验值

| 参数 | 典型范围 | 说明 |
|---|---|---|
| 耦合系数 α | 0.5 ~ 0.9 | 越大 Lss/Ltr 比值越高，但磁芯容易饱和 |
| 匝比 Naux/Nmain | 1:1 ~ 3:1 | ICL/HCL 中附加绕组与主绕组匝比 |
| Lss/Ltr 比值 | 3 ~ 7 | 相数越多比值越大 |
| 额外电感 Lc | nH ~ μH 级 | 控制耦合度的关键元件 |
| 工作频率 | 300kHz ~ 2MHz | 数据中心常用，移动设备更高 |
| 磁芯材料 | 铁氧体（DMR51）/ 金属粉芯 | 高频用铁氧体，大电流用粉芯 |

---

> 写到最后：耦合电感的核心思想就是**「花一个电感的钱，买到两个电感的好处」**——用磁耦合把稳态和动态解耦，让大电感和小电感同时存在。选型口诀：
>
> **相少用 DCL，相多用 ICL，都要好上 HCL。**
